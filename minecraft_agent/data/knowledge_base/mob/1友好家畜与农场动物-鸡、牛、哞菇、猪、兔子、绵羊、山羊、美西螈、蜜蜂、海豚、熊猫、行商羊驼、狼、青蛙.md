# 鸡
“ > 一种似乎总在观察的简单农场动物。鸡的脑袋里在想什么呢？它们在踱步的时候酝酿着什么计划呢？”

## 生成
### 自然生成
1. 鸡会自然生成于亮度等级9或以上（[Java版]）或7或以上（[基岩版]）的 [草方块] 上。
2. 在不同 [生物群系] 中，鸡会生成为不同的变种：寒带、热带或温带（默认）。这些变种影响其外观及产下的鸡蛋颜色。
3. 自然生成的鸡有5%的概率是幼年鸡。
### 人为生成
1. 掷出的 [鸡蛋] 击中物体破裂后，有1⁄8的概率生成1只幼年鸡；若此次生成成功，则有额外的1⁄32概率额外生成4只幼年鸡（即总共5只）。
### 鸡骑士
1. 幼年 [僵尸] 及其变种有小概率在生成时骑着鸡，形成 [鸡骑士]。
2. 鸡骑士的鸡不会下蛋，且被玩家杀死时掉落更多经验值。骑在上面的幼年僵尸死亡或离开后，鸡会恢复正常行为。

## 行为
1. 鸡会漫无目的地游荡，经常在水中游泳。它们从高处掉落时会扇动翅膀以减缓下落速度，避免摔伤。
2. 鸡会每隔5-10分钟（6000-12000游戏刻）下一个 [鸡蛋]。
3. 鸡会被 [豹猫] 和 [狐狸] 攻击，受到伤害时会快速逃跑。
4. 鸡可以被10格（[Java版]）或16格（[基岩版]）内手持 [小麦种子]、 [甜菜种子]、 [西瓜种子]、 [南瓜种子]、 [火把花种子] 或 [瓶子草荚果] 的玩家吸引。
5. 与其他动物不同，鸡会尝试跳上 [楼梯] 而不是走上去。

### 繁殖
1. 用上述任意种子喂食两只鸡，它们会进行 [繁殖]，生出一只幼年鸡，并掉落1–7经验值。繁殖后需等待5分钟（[Java版]）或1分钟（[基岩版]）才能再次繁殖。
2. 幼年鸡的变种从双亲的变种中随机选择。
3. 幼年鸡会在20分钟内长大，喂食种子可以加速其成长过程。

## 你知道吗
- 寒带鸡的设计灵感来源于现实中的波兰鸡。
- 鸡在空中被杀死时，其“尸体”下落速度较慢，且仍会显示翅膀扇动的动画。
- 由于尺寸小，鸡可以“藏”在 [漏斗] 或 [炼药锅] 中，从而避免受到上方熔岩或水的伤害。
- 游戏开发者Notch曾开玩笑说要将鸡改为鸭子，这引发了社区的热议，但最终被确认为一个玩笑。
- 在愚人节版本 [Java版2.0] 中，曾加入会下钻石和青金石的鸡变种。

# 牛
“ > 无论是需要皮革，填饱肚子，减轻痛苦，还是只是陪伴，牛都可以提供。”
**【牛（Cow）】** 是 [主世界] 中一种常见的友好生物。它们是 [皮革]、 [生牛肉]、 [牛排] 和 [奶桶] 的主要来源。

## 生成
### 变种
1. 牛会在 [草方块] 上至少有2格空间、亮度等级≥7的空间上生成。每只自然生成的牛有5%的概率生成为幼年个体。
2. 牛也能生成在 [村庄] 的牛圈里。
3. 在不同的 [生物群系] 中，牛会生成为不同的变种：
   *   **温带变种**：生成于除下表所列外的其他大多数生物群系。
   *   **寒带变种**：生成于冰雪、高山、针叶林、末地、下界及深暗之域等寒冷或恶劣生物群系。
   *   **热带变种**：生成于沙漠、热带草原、丛林、恶地、下界森林及红树林沼泽等炎热或繁茂生物群系。
### 哞菇
用 [剪刀] 为 [哞菇] 剪除蘑菇会使其变成牛。

### 铁桶与碗
1. 对任何哞菇使用 [铁桶] 可获得 [奶桶]。
2. 对任何哞菇使用 [碗] 可获得 [蘑菇煲]。
3. **棕色哞菇**在被喂食 [花] 后，下一次对其使用碗将获得 [谜之炖菜]（见下文）。
### 花与谜之炖菜
1. 给**棕色哞菇**喂食特定的 [小型花] 后，它会被“注入”该花对应的 [状态效果]。
2. 随后对其使用碗，会获得一碗带有该效果（持续时间为对应药水的一半）的 [谜之炖菜]，同时消耗掉花的效果。
3. 不同花朵提供的状态效果如下：
   - **[绒球葱]**：提供 [抗火] 效果。
   - **[蓝花美耳草]** 或 **[张开的眼眸花]**：提供 [失明] 效果。
   - **[兰花]** 或 **[蒲公英]**：提供 [饱和] 效果。
   - **[矢车菊]**：提供 [跳跃提升] 效果。
   - **[铃兰]**：提供 [中毒] 效果。
   - **[滨菊]**：提供 [生命恢复] 效果。
   - **[虞美人]** 或 **[火把花]**：提供 [夜视] 效果。
   - **[郁金香]**：提供 [虚弱] 效果。
   - **[凋灵玫瑰]**：提供 [凋零] 效果。
   - **[闭合的眼眸花]**：提供 [反胃] 效果。
4. 提供谜之炖菜后，哞菇会恢复为普通状态，再次使用碗只能获得蘑菇煲，除非重新喂食花。
### 繁殖
1. 哞菇会跟随手持 [小麦] 的玩家。

2. 发起冲刺需要目标与山羊之间至少有4格无障碍空间，且目标在山羊蓄力期间不移动。
3. 山羊快速冲向目标，并对撞到的生物造成近战伤害和较强的击退（成年9格，幼年4.5格）。被山羊冲撞的生物通常不会反击。

4. **尖叫山羊遗传**：繁殖一对普通山羊有2%的概率产下尖叫山羊；如有一方是尖叫山羊则概率约为50%；双方均为尖叫山羊则为100%\[仅 [Java版]\]或98%\[仅 [基岩版]\]。

对成年山羊使用 [铁桶] 可以挤奶，并获得 [奶桶]。

## 你知道吗
- 游戏中山羊的原型为现实中的 [洛基山羊]，而其音效取自现实中的 [家山羊]。
- 尖叫山羊并非游戏中虚构的变种，而是根据现实中某些山羊的尖叫行为设计的。
- 虽然山羊的冲撞具有伤害，但开发者表示，山羊的冲撞行为并非蓄意伤害，而是调皮的恶作剧。

2. 在 [Java版] 中，美西螈会受到具有 [穿刺] 魔咒的武器或 [三叉戟] 的额外伤害。

8. 失去螫针的蜜蜂仍可被激怒，但无法造成伤害。
9. 在 [Java版] 中，蜜蜂攻击玩家后，其螫针会短暂附着在玩家身上。
### 生物族群
1. 蜜蜂是 [节肢生物]，被具有 [节肢杀手] 魔咒的武器攻击时会受到额外伤害并被施加数秒的 [缓慢] IV效果。
### 繁殖
1. 蜜蜂会跟随手持花（闭合的眼眸花除外）、盛开的杜鹃花丛、樱花树叶等“合适食物”的玩家。
2. 对两只成年蜜蜂喂食合适的食物，它们会进入 [繁殖] 模式，生出一只幼年蜜蜂，并掉落1–7 [经验值]。繁殖后，双亲在5分钟\[仅 [Java版]\]/1分钟\[仅 [基岩版]\]内不能再次繁殖。
3. **例外**：喂食 [凋灵玫瑰] 会使蜜蜂获得 [凋零] 效果；喂食 [张开的眼眸花] 会使蜜蜂获得 [中毒] 效果；这两种物品**不能**用于繁殖。

5. 被激怒的蜜蜂仍然可以繁殖。

## 你知道吗
- 蜜蜂的体型被设计为近半个方块大小，因为开发团队认为这个尺寸“最可爱”。
- 蜜蜂攻击后失去螫针会死亡的设定参考了现实：现实中的工蜂蜇刺后内脏会被拉出导致死亡。

2. **在 [基岩版] 中，杀死驯服的狼不会掉落经验。**
3. 杀死幼年狼不会掉落经验。

## 行为
1. **状态：**
    *   **野生：** 尾巴抬起约36°，双眼一白一黑像素。
    *   **愤怒：** 受到攻击或发现敌对目标时，双眼发红，尾巴抬起约88.2°。此时无法被驯服。
    *   **驯服：** 被玩家驯服后，脖子上出现项圈，尾巴抬起角度随生命值变化。
2. **移动与跟随：** 驯服的狼站立时会跟随主人（10格外游荡跟随，12格外直接走向或传送）。对其按下使用键可使其坐下，坐下时不会跟随。
3. **传送：** 站立且未被拴住的驯服狼，在距离主人超过12格时会尝试传送至主人附近。但若其正坐在矿车、船上或被拴住，则不会传送。
4. **乞求：** 玩家在8格内手持肉类或骨头时，狼会侧头乞求食物。

### 攻击
1. 狼会主动攻击 [骷髅]、 [凋灵骷髅]、 [流浪者]、 [沼骸]、[焦骸]、 [兔子]、 [狐狸]、 [绵羊] 和幼年 [海龟]。
2. 驯服的狼（站立时）会攻击伤害其自身或主人的生物（除部分豁免目标，如主人的其他宠物）。
3. **骷髅、狐狸、兔子等会主动避开狼。** [羊驼] 和 [行商羊驼] 会向野生狼吐唾沫。
4. **狼永远不会攻击 [苦力怕] 和 [恶魂]。**
5. 狼会跳跃攻击，但空中不会造成伤害。

### 驯服与喂食
1. 对野生狼使用 [骨头] 可尝试驯服，每次使用有1⁄3的概率成功，平均需要3根骨头。
2. 驯服后，可使用 [染料] 改变项圈颜色。
3. 驯服的狼生命值可通过喂食**肉类**恢复，恢复量为该食物为玩家恢复的 [饥饿值] 的两倍。
    *   可使用 [腐肉] 和 [生鸡肉] 等，**不会**产生负面效果。
    *   在 [Java版] 中，喂食 [兔肉煲] 会返还一个 [碗]。
4. 驯服的狼可装备 [狼铠] 以提供11护甲值，使用 [犰狳鳞甲] 可修复狼铠，使用 [剪刀] 可卸下狼铠。
5. **狼铠能抵消大多数类型的伤害**，但无法抵消 [魔法伤害]、 [凋零]、 [溺水]、 [饥饿]、 [虚空] 等特殊伤害。

### 繁殖
1. 对两只生命值已满的驯服狼喂食（除部分鱼类和兔肉煲外）可使其进入“ [求爱模式]”，繁殖出幼年狼。繁殖后5分钟\[Java版\]/1分钟\[基岩版\]内无法再次繁殖。
2. 繁殖出的幼年狼**天生已被驯服**，项圈颜色为父母项圈的有效混色（若无有效混色则随机继承一方）。
3. 对幼年狼喂食可加快其生长，每次喂食减少约10%的剩余成长时间。

## 你知道吗
- 狼会远离水或雨后甩动身体，甩出水花粒子。
- 驯服的狼死亡时会向主人\[仅 [Java版]\]/所有玩家\[仅 [基岩版]\]显示死亡消息。
- 狼无法通过攻击来推进“冒险”、“瞄准目标”等需要伤害生物的进度，但被狼杀死可以完成“冒险”进度。
- 狼的多种毛色变种可能参考了现实中的动物，如灰狼似哈士奇，斑点狼似非洲野犬，条纹狼似土狼或条纹鬣狗，雪狼似北极狼。
- 狼铠有独特的破损纹理，会随着耐久度降低而逐渐开裂。

## 生成
### 自然生成
1. 温带青蛙自然生成于 [沼泽] 生物群系，热带青蛙自然生成于 [红树林沼泽] 生物群系。
2. 在 [基岩版] 中，青蛙生成无亮度要求，空间检测范围为2×2×2。
### 蝌蚪
[蝌蚪] 会在生成20分钟后成长为青蛙，这是生存模式中寒带青蛙唯一的生成途径。给其喂食 [黏液球] 可加速成长，每次缩短10%的剩余成长时间。
### 肤色
青蛙有寒带、热带、温带3种变种，分别为绿色、白色、橙色，青蛙的变种取决于其生成时（包括蝌蚪成长为青蛙）所处的生物群系。具体对应关系如下：
- **寒带青蛙**：生成于冰雪、针叶林、冷水海洋、末地及深暗之域等寒冷生物群系。
- **温带青蛙**：生成于河流、平原、森林、山地、大部分海洋及洞穴等温带生物群系。
- **热带青蛙**：生成于丛林、沙漠、热带草原、红树林沼泽、暖水海洋及下界等炎热生物群系。

## 掉落物
青蛙被 [玩家] 或驯服的 [狼] 杀死时会掉落1–3 [经验值]。

## 行为
1. 青蛙在陆地上会漫无目的地缓慢游荡、四处跳跃，在水中则会游泳。在水中，青蛙的移动速度要快得多，且通常会向上游动并停留在水面上。
2. 青蛙最高可以跳8格高。与 [山羊] 相似，青蛙在摔落时受到的伤害相较正常水平少5（）。青蛙会倾向于跳上 [大型垂滴叶] 和 [睡莲]。
3. 青蛙可以被 [拴绳] 拴住，且不会溺水。
### 捕食
1. 青蛙会捕食 [小型史莱姆]。它们发现目标时会提高移动速度，伸出舌头迅速将其吸入口中，并掉落1个 [黏液球]。
2. 青蛙也会以相同的方式捕食 [小型岩浆怪]。寒带、温带、热带的青蛙吞食岩浆怪后分别会掉落1个青翠、赭黄和珠光变种的 [蛙明灯]。
3. 在 [Java版] 中，青蛙捕食时会对猎物造成10（）的基础攻击伤害直到其死亡。在 [基岩版] 中，被青蛙吞食的实体会瞬间消失。
### 繁殖
1. 青蛙会跟随10格内\[仅 [JE]\]/16格内\[仅 [BE]\]手持 [黏液球] 的玩家。
2. 给两只青蛙喂食黏液球会使其 [繁殖]，掉落1–7 [经验值]。繁殖5分钟\[仅 [Java版]\]/1分钟\[仅 [基岩版]\]后才能再次繁殖。
3. 其中一只青蛙会搜寻上方有空气的 [水源]，并在其上方产下 [青蛙卵]。

## 你知道吗
- 热带青蛙的原型为现实中的非洲 [大灰攀蛙]，温带青蛙的原型为现实中的 [牛蛙]。
- 青蛙最初被设计为吃掉萤火虫来产生 [蛙明灯]，但在开发者发现一些种类的萤火虫在现实生活中对一些种类的青蛙有毒后，这项计划就被取消。
- 青蛙和 [美西螈] 都是两栖动物，但是青蛙离开 [水] 后不会因为脱水而受到伤害并死亡。
