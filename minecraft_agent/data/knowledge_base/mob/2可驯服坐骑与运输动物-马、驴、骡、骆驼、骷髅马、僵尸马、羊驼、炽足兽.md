# 马
“ > 马是一个出自于Minecraft野外的平静而自信的同胞。它们三五成群，在平原和热带草原生物群系上漫步、低头吃草、并嗖嗖地甩着尾巴。但是这些马不是笨拙、不起眼的生命，它们是一种骄傲而敢于冒险的生物。”

## 生成
1. 马会在 [平原]、[向日葵平原]、[热带草原]、[热带高原] 和 [风袭热带草原] 生物群系自然生成。
2. 马可以在 [村庄] 的马厩和动物围栏中生成。
3. 每匹马都有20%的概率生成为幼年个体。

## 掉落物
下表展示了在不同等级的抢夺附魔影响下，击杀马时掉落皮革的数量范围、掉落概率及平均数量。
**【皮革】**
**无抢夺魔咒**：
1. 数量范围：0–2 个
2. 掉落概率：66.67%
3. 平均数量：1.00 个
**抢夺 I**：
1. 数量范围：0–3 个
2. 掉落概率：83.33%（Java版） / 66.67%（基岩版）
3. 平均数量：1.50 个（Java版） / 1.33 个（基岩版）
**抢夺 II**：
1. 数量范围：0–4 个
2. 掉落概率：91.67%（Java版） / 66.67%（基岩版）
3. 平均数量：2.00 个（Java版） / 1.67 个（基岩版）
**抢夺 III**：
1. 数量范围：0–5 个
2. 掉落概率：94.44%（Java版） / 66.67%（基岩版）
3. 平均数量：2.50 个（Java版） / 2.00 个（基岩版）

**【总结说明】**
版本差异：在 [Java版] 中，抢夺附魔能显著提高皮革的掉落概率及平均数量；而在 [基岩版] 中，抢夺附魔仅增加皮革的最大掉落数量，不改变其掉落概率。
额外掉落：击杀幼年马不会掉落任何物品或经验。马死亡时会掉落其装备的 [鞍] 和 [马铠]。
经验：被 [玩家] 或驯服的 [狼] 杀死时，掉落1–3 [经验值]。

## 行为
马会漫无目的地四处游荡、时不时鸣叫、摇尾巴以及低头吃草（不会破坏草皮）。马可以缓慢地恢复损失的生命值。
### 驯服
1. 成年的马可以被驯服。对着未驯服的马按下使用键可尝试骑乘，若驯服失败，马会将玩家甩下并增加好感度。
2. 驯服成功率取决于`Temper`值（好感度）。好感度可通过骑乘失败（增加5点）或喂食来提升。
3. 当马不再试图甩下玩家并产生红心粒子时则驯服成功。
### 装备
1. 驯服的马可以装备 [鞍] 和 [马铠]。
2. 装备鞍后，玩家才能控制骑乘。玩家可使用 [剪刀] 移除鞍或马铠。
### 骑乘
1. 骑上装备鞍的马后，玩家能用方向键、跳跃等控制其移动，并可以安全通过至少2.75格高的空间。
2. 每匹马的**移动速度**和**跳跃力量**各不相同（详见下文统计部分）。其速度和跳跃高度受相关状态效果影响。
3. 骑马时，玩家可以正常使用物品、与方块或生物交互。

### 喂食
对马使用有效的食物可以治疗、加速幼年马成长、增加好感度或促使其繁殖。
下表列出了各种食物对马的影响。
**【糖】**
1. 恢复生命值：1（）
2. 缩短幼年马成长时间：30秒
3. 增加好感度：+3
**【小麦】**
1. 恢复生命值：2（）
2. 缩短幼年马成长时间：20秒
3. 增加好感度：+3
**【苹果】或【胡萝卜】**
1. 恢复生命值：3（）
2. 缩短幼年马成长时间：1分钟
3. 增加好感度：+3
**【金胡萝卜】**
1. 恢复生命值：4（）
2. 缩短幼年马成长时间：1分钟
3. 增加好感度：+5
**【金苹果】或【附魔金苹果】**
1. 恢复生命值：10（）
2. 缩短幼年马成长时间：4分钟
3. 增加好感度：+10
**【干草捆】**
1. 恢复生命值：20（ × 10）
2. 缩短幼年马成长时间：3分钟
3. 增加好感度：无

- **治疗**：给生命值未满的马喂食可恢复生命值。
- **加速成长**：给幼年马喂食可缩短其成长时间。
- **进入繁殖模式**：给生命值已满的驯服成年马喂食 [金胡萝卜]、 [金苹果] 或 [附魔金苹果] 可使其进入求爱模式。

### 繁殖
1. 对两头生命值已满的驯服成年马使用 [金胡萝卜]、 [金苹果] 或 [附魔金苹果] 可使它们进入 [繁殖] 模式，并产下一匹幼年马。
2. 对一匹马和一头驴使用上述食物，可繁殖出 [骡]。
3. 繁殖后会掉落1–7 [经验值]，双亲会进入繁殖冷却时间（Java版5分钟/基岩版1分钟）。
4. 幼年马需要20分钟（现实时间）成长为成年个体，喂食可以加速此过程。
5. 后代的**毛色与花纹**通常继承自父母之一，但有较小概率发生随机突变。
6. 后代的**速度、跳跃和生命值**等统计数值继承自父母，并可能发生变异（具体机制因版本而异）。

## 你知道吗
- 马有7种毛色（白、淡栗、深枣红、褐、黑、灰、深褐）和5种花纹（无、长袜白斑、雪片白斑、白斑点、黑斑点），共35种外观组合。
- 马的模型设计得到了模组“Mo' Creatures”作者Dr.Zhark的协助。
- 与驴和骡不同，马可以装备 [马铠] 来获得额外的护甲值。
- 马的移动速度和跳跃能力是随机的，玩家需要寻找或繁殖出属性优良的马匹。

# 骆驼
“ > 在足迹与故事更新中，你会发现沙漠里的一位新居民——骆驼！骆驼也许以暴躁而闻名，但实际上它们在Minecraft中却是相当温顺的存在。它们喜欢坐下和漫无目的地闲逛，但它们也同样喜欢去冒险。”
**【骆驼（Camel）】** 是一种生成于 [沙漠] 和 [沙漠村庄] 的 [友好生物]。

## 目录
- [1生成]
- [2掉落物]
- [3行为]
  - [3.1骑乘]
  - [3.2喂食与繁殖]
- [4你知道吗]

## 生成
1. 骆驼会自然生成于 [沙漠] 生物群系。在 [基岩版] 中，仅生成在露天的沙子、红沙、砂岩上方。
2. 骆驼也会自然生成于 [沙漠村庄] 的中心，每个沙漠村庄只会生成一只骆驼。

## 掉落物
- 成年骆驼被 [玩家] 或驯服的 [狼] 杀死时掉落1–3 [经验值]。
- 套在骆驼身上的 [鞍] 也会在其死亡时掉落。
- 幼年骆驼不掉落任何物品和经验。

## 行为
骆驼会漫无目的地游荡，也会躲避 [水]、 [熔岩] 和悬崖。骆驼可以缓慢地恢复生命值。
1. **坐下**：骆驼在未被骑乘时有时会坐下歇息。坐着的骆驼不会被其他生物推动。若被骑乘、受到伤害或被拴绳拉开时会迅速站起。
2. **抗摔落**：骆驼从小于6.5格高的方块跌落时不会受到 [摔落伤害]。
### 骑乘
1. 玩家可以为骆驼装备 [鞍]。装备鞍后，玩家可控制其移动。
2. 骆驼的**独特能力**：按下跳跃键可以进行冲刺，正常情况下最多向前冲刺12格，并有向上跳跃的效果。冲刺后有2.75秒的冷却时间。
3. **双人骑乘**：一只骆驼可以搭载两名玩家。前座玩家控制方向，后座玩家可以转动身体并与周围互动。来自低处（低于玩家0.5格以上）的生物近战攻击无法伤害骑乘者。
4. 在 [Java版] 中，骑乘着坐下的骆驼时，按下向前键可以使骆驼站起。
### 喂食与繁殖
1. **跟随**：骆驼会跟随10格内\[仅 [Java版]\]/16格内\[仅 [基岩版]\]手持 [仙人掌] 的玩家。
2. **喂食效果**：
   *   给成年骆驼喂食仙人掌可恢复2（）点生命值。
   *   给幼年骆驼喂食仙人掌可缩短其成长时间。
3. **繁殖**：给两只生命值已满的成年骆驼喂食仙人掌能让它们 [繁殖] 出一只幼年骆驼，并掉落1-7个经验球。双亲会进入繁殖冷却时间（Java版5分钟/基岩版1分钟）。

## 你知道吗
- Minecraft中的骆驼是基于现实中的 [单峰骆驼] 开发的，这种骆驼背上只有一个驼峰。
- 骆驼是游戏中唯一可以**同时搭载两名玩家**的陆地坐骑。
- 其高大体型可以有效防御来自地面 [尸壳] 等生物的近战攻击。

1. 成年羊驼可以被驯服。反复骑乘羊驼直到出现心形 [粒子] 即为驯服成功。驯服难度取决于羊驼的“好感度”。
2. 已驯服的羊驼无法装备 [鞍]，因此玩家无法控制其移动。
3. 已驯服的羊驼可以装备一个 [地毯] 进行装饰。使用 [剪刀] 可以剪下地毯。
4. 手持 [箱子] 对已驯服的羊驼使用，可以为其装备箱子。箱子的物品栏格数（3、6、9、12或15格）由羊驼的**强度**值决定（格数 = 3 × 强度）。装备后无法卸下，除非羊驼死亡。
### 喂食与繁殖
1. 给羊驼喂食可以恢复其生命值、增加好感度（使其更容易被驯服）并缩短幼年个体的成长时间。
   * 喂食 [小麦] 恢复 2（）生命值，增加3点好感度。
   * 喂食 [干草捆] 恢复 10（）生命值，增加6点好感度，并能使已驯服的羊驼进入 [繁殖] 模式。
2. 给两只生命值已满的已驯服羊驼喂食干草捆，可以使它们繁殖，产下一只幼年羊驼。
3. 幼年羊驼的毛色会等概率地遗传自双亲之一。其**强度**值在1和较强双亲的强度值之间随机选取，并有3%的概率额外+1（最大为5）。

## 你知道吗
- 羊驼的绿色和紫色地毯样式上分别有 [苦力怕] 与 [末影人] 的图案。灰色地毯的样式类似超级英雄面具。
- 在 [Java版] 中，拥有 [隐身] 效果的羊驼仍会显示其装备的地毯和箱子。
- 羊驼有四种自然的毛色：沙褐色（默认）、白色、棕色和灰色。

# 炽足兽
“ > 炽足兽是下界中独一无二的摆渡者。玩家可通过类似骑猪的方式把炽足兽当做坐骑，以跨越下界中广阔的熔岩海。我个人不介意偶尔花三个小时进行跨海架桥工程，也不在乎失足跌入两千多度高温的液体中洗个澡，但炽足兽确实能让我更轻松地渡过熔岩。你只需要一个鞍和一把诡异菌钓竿，就能驾驭它！”
**炽足兽（Strider）** 是 [下界] 内一种可骑乘的友好生物，能在 [熔岩] 上行走而不受伤害。

## 生成
炽足兽会生成于下界的 [熔岩] 中，熔岩的高度至少为2格\[仅 [Java版]\]或1格\[仅 [基岩版]\]，且熔岩上方必须有空气\[仅 [Java版]\]。炽足兽每400 [游戏刻]（20秒）会尝试生成一次。幼年炽足兽有10%的概率会生成在成年炽足兽头上。
在非和平 [难度] 下， [僵尸猪灵] 有三十分之一的概率会骑在炽足兽头上。这种情况下炽足兽会装备鞍，且僵尸猪灵会手持 [诡异菌钓竿]。在 [Java版] 中，炽足兽头上的僵尸猪灵不会自然消失。在和平难度下，僵尸猪灵骑乘炽足兽的组合仍会尝试生成，但僵尸猪灵会消失，只留下装备鞍的炽足兽。

## 掉落物
幼年炽足兽死亡时不会掉落任何物品和经验。
成年炽足兽 [死亡] 时会掉落：
蜘蛛在被杀死时会掉落线。然而，抢夺附魔对线掉落的影响在Java版和基岩版中有显著差异。
一.【Java版】:
在【Java版】中，抢夺附魔能有效增加蜘蛛的线掉落数量。
【无抢夺魔咒】：
1. 数量范围： 2–5 根线
2. 掉落概率： 100%
3. 平均数量： 3.50 根
4. 分布： 掉落2、3、4、5根线的概率均等，各为25%。
【抢夺 I】：
1. 数量范围： 2–6 根线
2. 平均数量： 4.00 根
3. 分布： 掉落数量上限提升至6根，概率分布发生变化。
【抢夺 II】：
1. 数量范围： 2–7 根线
2. 平均数量： 4.50 根
3. 抢夺 III：
4. 数量范围： 2–8 根线
5. 平均数量： 5.00 根
总结 (Java版): 随着抢夺附魔等级提升，线的最大掉落数量和平均掉落数量稳定增加。
二.【基岩版】:
在基岩版中，抢夺附魔对蜘蛛的线掉落没有任何影响。
【所有抢夺等级 (无抢夺 至 抢夺III)】：
1. 数量范围： 2–5 根线
2. 掉落概率： 100%
3. 平均数量： 3.50 根
4. 分布： 无论是否使用抢夺附魔，掉落2、3、4、5根线的概率始终均等，各为25%。
总结 (基岩版): 抢夺附魔无法增加蜘蛛的线掉落。
三.【通用掉落物】
【鞍】： 如果炽足兽身上装备了鞍，那么杀死它时会掉落该鞍。
【经验值】：
被玩家或驯服的狼杀死时，蜘蛛会掉落经验值。
【Java版】： 掉落1–2点经验值。
【基岩版】： 掉落1–3点经验值。
【核心版本差异】
关键区别： 抢夺附魔是否能增加蜘蛛的线掉落数量。
Java版：可以。抢夺等级直接影响掉落数量上限和平均值。
基岩版：不可以。线掉落数量固定，不受抢夺影响。

## 行为
1. 与其他 [生物] 不同，炽足兽可在 [熔岩] 顶部行走。它们也会主动走向熔岩。炽足兽免疫火焰伤害，但在 [基岩版] 中，幼年炽足兽待在装有熔岩的 [炼药锅] 中会着火。
2. 炽足兽浸入熔岩后会缓慢上升到熔岩顶部。在 [Java版] 中，炽足兽在水中会下沉。
3. 炽足兽接触 [水] 或 [雨] 时每0.5秒会受到1（）的伤害。即使在熔岩中，炽足兽也依然会受到来自雨的伤害。它们在接触 [喷溅型水瓶] 和 [滞留型水瓶] 时每次也会受到1（）的伤害。但 [雪球] 或 [降雪] 不会伤害它们；幼年炽足兽站在装满水的炼药锅中也不会受到伤害。在 [细雪] 中进入冰冻状态的炽足兽每2秒受到5（）伤害，这是多数生物所受伤害的5倍。
4. 离开熔岩的炽足兽会打寒颤并且颜色发紫，但不会因此受到伤害。它们在熔岩外移动速度会变慢。
5. 在 [Java版] 中，被 [僵尸猪灵] 骑乘的炽足兽会在僵尸猪灵被激怒后追赶激怒僵尸猪灵的生物。
生成在成年炽足兽头上的幼年炽足兽仍会成长为成年炽足兽，但它们长大后不会掉下来。对成年炽足兽头上的炽足兽使用 [拴绳] 可强制将其拉下来。
6. 在 [基岩版] 中，炽足兽在熔岩上行走时，其头上的炽足兽也是红色纹理，而不是紫色。
炽足兽会跟随10格\[仅 [JE]\]或16格\[仅 [BE]\]内手持 [诡异菌] 或 [诡异菌钓竿] 的玩家。
7. 在 [基岩版] 中，炽足兽陷入细雪时会保持红色纹理，直到冰冻计时结束开始受到伤害，才会恢复紫色。在该过程中，炽足兽不会因细雪而颤抖。

### 骑乘
骑乘炽足兽的玩家：
1. 玩家可对成年炽足兽使用 [鞍] 使其可被骑乘，并且可使用 [拴绳] 牵引。类似用 [胡萝卜钓竿] 控制 [猪] 的前进方向一样，炽足兽可使用 [诡异菌钓竿] 来控制方向。
2. 炽足兽在熔岩上只能走上与熔岩高度相同的方块，若该方块高于熔岩，炽足兽则无法上岸。炽足兽在陆地上会自动走过1格高的方块。
速度计算
3. 炽足兽在熔岩上的常规行进速度为4.24米/秒\[仅 [Java版]\]或3.71米/秒\[仅 [基岩版]\]，在陆地上的常规行进速度为1.78米/秒\[仅 [Java版]\]或1.64米/秒\[仅 [基岩版]\]。
4. 使用诡异菌钓竿可以加快炽足兽的行进速度，同时使诡异菌钓竿降低1点耐久度。此时炽足兽会获得一个7到49秒\[仅 [Java版]\]或16秒\[仅 [基岩版]\]的加速时间，加速期间的速度会在常规行进速度的基础上加速至1+1.15×sin⁡(tt0π)倍，其中t是已经过的加速时间，t0是总加速时间。

### 繁殖
炽足兽可通过喂食诡异菌 [繁殖]，同时会掉落1–7 [经验值]。繁殖后双亲有5分钟\[仅 [JE]\]/1分钟\[仅 [BE]\]的冷却时间。繁殖出的幼年个体需20分钟才能成长为成年个体。幼年个体可使用诡异菌加速成长，每次减少约10%剩余成长时间。

## 进度
以下是Java版中两个关于驾驭炽足兽的进度，它们是“下界”进度分支的一部分。
【画船添足 (This Boat Has Legs)】
1. 游戏内描述： 手持诡异菌钓竿骑乘炽足兽。
2. 上游进度： 【下界】
3. 实际需求： 玩家需要拿着诡异菌钓竿并骑乘上炽足兽，然后按下使用键（右键） 来加速炽足兽，此操作必须成功消耗诡异菌钓竿的耐久度才算完成。仅仅手持钓竿骑上去是不够的。
【温暖如家 (Feels Like Home)】
1. 游戏内描述： 带炽足兽在主世界的熔岩湖上来一场长——途旅行。
2. 上游进度： 【画船添足】
3. 实际需求： 在完成“画船添足”进度后，玩家需要骑着炽足兽，在主世界的熔岩上，单向连续移动50个方块的距离。
【总结说明】：
进度关系： “画船添足”是基础进度，而“温暖如家”是其直接下游的进阶进度。玩家必须先学会如何用诡异菌钓竿正确驾驭炽足兽，才能带它进行长途旅行。
核心机制： 这两个进度引导玩家熟悉炽足兽的整套操控机制：首先用诡异菌引诱并骑乘，然后用诡异菌钓竿控制方向与加速，最后利用它能在熔岩上行走的特性进行跨维度移动。
关键区别： “画船添足”进度关键在于使用诡异菌钓竿加速这一动作，而“温暖如家”进度关键在于在特定环境（主世界熔岩） 完成特定距离（50格） 的移动。

## 你知道吗
- 炽足兽的设计共迭代出了4个版本：
  - 最初的设计名为“Crawler”，取“熔岩中匍匐穿行”之意，参考了 [水熊虫]。
  - 第二个设计在外形上参考了 [恶魂]，有多条长腿，使之能触及熔岩海的底部行走。然而模型过长易出现技术性问题。
  - 第三个设计参考了鱼类，长有鳍，能在熔岩中游泳。因开发者认为不够“异样”（alien）而被弃用，但“鳍”的纹理曾出现在游戏文件中。
  - 最终被加入游戏中的设计参考了能在水面上行走的 [水黾]（Waterstrider）。
- [Brandon Pearce] 曾打算给炽足兽加入离开熔岩后会冻伤的特性。它们离开熔岩后会缩起双腿，并且像离开水后的 [鱼] 一样四处跳动直至死亡，最后这一想法由于“过于悲哀”而被舍弃了。
- 炽足兽被弃用的特性还包括：
  - 独有的盔甲和特殊的鞍。
  - 有助于采食 [诡异菌]的长舌头，吃 [TNT] 时会爆炸，吃下 [剑] 会受到伤害。
- 炽足兽行走时左右摆动的动画效果其实是在开发过程中无意间出现的，Brandon觉得这“离奇而滑稽”，因此将其保留了下来。
- 炽足兽死亡时掉落的 [线] 是它们的鬃毛，在设定上是 [猪灵] 合成 [弩] 所需线的来源。
