# 游戏模式
Minecraft 包含多种游戏模式，适合不同类型的玩家。

【生存模式（Survival）】
玩家需要自行采集资源、合成物品、建造庇护所，并在夜间抵御怪物攻击。生命值和饥饿值是核心生存指标。

【创造模式（Creative）】
玩家拥有无限方块和资源，可以飞行，且无法受到伤害。主要用于建筑创作和地图制作。

【冒险模式（Adventure）】
玩家无法直接破坏方块（除非手持对应工具），适用于自定义地图和冒险关卡。

【旁观模式（Spectator）】
玩家以幽灵状态穿越世界，无法交互，常用于观察和调试。

【极限模式（Hardcore）】
与生存模式相同，但死亡后存档将被删除（或锁定），极具挑战性。仅 Java 版可用。

# 世界生成与维度
Minecraft 世界由种子（Seed）决定，相同种子会生成相同的地形。游戏包含三个主要维度。

## 主世界
主世界是玩家默认出生的维度，拥有丰富多样的地形和生物群系。世界在水平方向上理论上无限延伸，垂直方向从 Y=-64 到 Y=320（Java版1.18+）。

【地形层次】
1. 深层地下（Y=-64 ~ Y=0）：深板岩层，含有深层矿石，生成深暗之域。
2. 地下层（Y=0 ~ Y=64）：普通矿石丰富区，含熔岩湖。
3. 地表层（Y=60 ~ Y=100）：大多数生物群系的地表，建筑主要区域。
4. 山地层（Y=100 ~ Y=256）：极端山地、雪山等高海拔地形。
5. 山峰层（Y=256 ~ Y=320）：仅极端山峰生物群系到达此高度。

## 下界
下界通过 [下界传送门]（由黑曜石框架点燃）进入，是一个充满熔岩与危险的维度。

【下界特性】
1. 下界中1格水平距离 = 主世界8格水平距离，是快速旅行的重要途径。
2. 下界没有昼夜交替。
3. 水在下界会立刻蒸发，无法放置水桶。
4. 床在下界使用会爆炸，造成巨大伤害。
5. 下界顶部（Y=128以上）是坚固的岩浆基岩层，可用于建造安全的高速公路。

【下界生物群系】
- 下界荒地：最常见的下界区域，布满砂砾、灵魂砂和岩浆。
- 绯红森林：红色菌类密布，生成猪灵和疣猪兽。
- 诡异森林：蓝绿色调，生成骷髅。
- 灵魂砂谷：灵魂砂与灵魂土覆盖地面，生成骷髅和烈焰人。
- 玄武岩三角洲：布满玄武岩柱，生成岩浆怪。

## 末地
末地通过击败末影龙后开启的 [末地传送门] 进入（也可在要塞中找到传送门）。

【末地特性】
1. 中央岛屿是末影龙的战斗场所，上有末地水晶为龙恢复血量。
2. 击败末影龙后，外岛传送门开启，可通过末地折跃门到达外岛。
3. 外岛有末地城市，内含鞘翅（Elytra）和丰富战利品。
4. 末影珍珠在末地中极为有用，可快速移动。


# 生物群系
生物群系（Biome）决定了地形、植被、生物种类和气候特征。

## 陆地生物群系

【平原（Plains）】
1. 特征：平坦开阔，草地覆盖，偶有树木。
2. 生成于：几乎与所有生物群系相邻，尤其常见于森林与沙漠之间。
3. 特产：马、驴、牛、羊大量生成；村庄极为常见。

【森林（Forest）】
1. 特征：橡木和白桦树密集分布。
2. 生成于：常与平原、桦木森林、黑森林相邻。
3. 特产：木材丰富，蘑菇常见，有时生成花朵。

【桦木森林（Birch Forest）】
1. 特征：白桦树为主，地面平坦。
2. 生成于：常与森林、黑森林过渡相邻。
3. 特产：白桦木材，适合建筑使用。

【黑森林（Dark Forest）】
1. 特征：橡木和深色橡木密集，林冠遮挡阳光，即使白天也可生成怪物。
2. 生成于：常与森林、桦木森林、沼泽相邻。
3. 特产：林地府邸（Woodland Mansion）仅生成于此。

【针叶林（Taiga）】
1. 特征：云杉树为主，地面有蕨类植物。
2. 生成于：常与平原、雪地、山地相邻。
3. 特产：狼（可驯服）、兔子、浆果丛生成。

【雪地针叶林（Snowy Taiga）】
1. 特征：积雪覆盖的云杉森林。
2. 生成于：靠近雪地平原与针叶林之间。
3. 特产：与针叶林相同，但气候更寒冷，雪狐在此生成。

【雪地平原（Snowy Plains）】
1. 特征：完全被积雪覆盖的平坦地形，水面结冰。
2. 生成于：常与雪地针叶林、冰刺平原相邻，远离沙漠和热带。
3. 特产：北极熊、雪狐生成；村庄为雪地变种。

【冰刺平原（Ice Spikes）】
1. 特征：巨大的蓝色冰刺柱从地面伸出，极为壮观。
2. 生成于：仅生成于雪地平原内部，是其变种。
3. 特产：堆积冰，可用丝绸之触镐获取冰块。

【沙漠（Desert）】
1. 特征：沙子覆盖地表，无草地，不下雨，日夜温差大。
2. 生成于：常与稀树草原、恶地相邻，远离雪地和森林。
3. 特产：沙漠庙（Desert Temple）和沙漠村庄；仙人掌、甘蔗生成；僵尸村庄较常见。

【稀树草原（Savanna）】
1. 特征：热带草原，金合欢树（Acacia）生长，不下雨。
2. 生成于：常与沙漠、平原、热带雨林相邻。
3. 特产：金合欢木（橙色木材），村庄常见；马、驴在此出现。

【热带雨林（Jungle）】
1. 特征：高大的热带树木，竹子（1.14+），地面覆盖热带草。
2. 生成于：常与稀树草原、蘑菇岛、沼泽相邻。
3. 特产：可可豆、竹子、热带鱼、丛林庙（Jungle Temple）；豹猫和鹦鹉在此出现。

【沼泽（Swamp）】
1. 特征：低洼湿地，橡木上有藤蔓，地面积水，色彩偏绿。
2. 生成于：常与平原、黑森林、热带雨林相邻。
3. 特产：女巫小屋（Witch Hut）；青蛙、史莱姆在此夜间生成；睡莲常见。

【红树林沼泽（Mangrove Swamp）】（1.19+）
1. 特征：红树（Mangrove）密集，地面为泥土，水面下有根系。
2. 生成于：常与温暖沼泽相邻，温暖区域的沿海。
3. 特产：红树木材，泥土方块（可合成泥砖），青蛙大量生成。

【山地（Mountains/Windswept Hills）】
1. 特征：高耸山峰，石头地表，山顶有积雪。
2. 生成于：可与多种生物群系毗邻，常处于平原、针叶林边缘。
3. 特产：高度较高，矿石暴露概率增加；骡子和美洲驼在此生成。

【樱花树林（Cherry Grove）】（1.20+）
1. 特征：粉色樱花树密布，花瓣飘落，极为美丽。
2. 生成于：常与草地山地相邻，山地区域的变种。
3. 特产：樱花木材（粉色系），猪在此生成。

【恶地（Badlands）】
1. 特征：红色、橙色、黄色的彩色陶土地形，峡谷密布。
2. 生成于：常与沙漠、稀树草原相邻，极干旱区域。
3. 特产：大量彩色陶土、红沙、仙人掌；金矿石在恶地地表附近额外大量生成；废弃矿井常暴露在峡谷壁上。

## 水域生物群系

【海洋（Ocean）】
1. 特征：大面积水体，水下地形多样。
2. 变种：温暖海洋（珊瑚礁）、寒冷海洋、冰冻海洋（冰山）、深海。
3. 特产：海底神殿仅在深海生成；各类海洋生物；海草、珊瑚。

【河流（River）】
1. 特征：蜿蜒穿过其他生物群系的水道，通常较浅。
2. 生成于：可切割任何生物群系，是生物群系之间的天然分界线。
3. 特产：甘蔗、黏土常见；鱿鱼和鲑鱼在此生成。

【蘑菇岛（Mushroom Fields）】
1. 特征：紫色菌土覆盖，巨型蘑菇遍布，极为罕见。
2. 生成于：通常独立存在于海洋中，不与陆地生物群系直接相邻。
3. 特产：唯一不生成任何敌对怪物的生物群系（除蝙蝠外）；哞菇（Mooshroom）仅在此生成。

## 地下与特殊生物群系

【深暗之域（Deep Dark）】（1.19+）
1. 特征：Y=0以下的深层地下，深板岩和幽匿块覆盖地面，极为黑暗。
2. 生成于：深层地下，Y=-64 ~ Y=0 之间，通常需要深度挖掘才能到达。
3. 特产：古城（Ancient City）仅生成于此；监守者（Warden）在此出没；幽匿系列方块（幽匿块、幽匿感测体、幽匿尖啸体）。

【溶洞（Lush Caves）】（1.18+）
1. 特征：地下洞穴中覆盖滴水石、苔藓、发光浆果，色彩丰富。
2. 生成于：地下洞穴系统中，表面通常上方有茂盛的草地或热带雨林。
3. 特产：滴水石（Stalactite/Stalagmite）、发光浆果（Glow Berries）；水晶虾在此游动。

【滴水石洞穴（Dripstone Caves）】（1.18+）
1. 特征：大量石笋和钟乳石，偶有岩浆湖。
2. 生成于：地下洞穴系统中，与普通洞穴相互连接。
3. 特产：大量滴水石块（可制作为锋利武器或熔炉燃料）。

## 生物群系过渡规律

Minecraft 的生物群系生成遵循温度和湿度梯度，相似气候的生物群系倾向于相邻生成：

【温度梯度（从冷到热）】
雪地平原 → 冰刺平原 → 针叶林 → 雪地针叶林 → 平原 → 森林 → 桦木森林 → 黑森林 → 稀树草原 → 沙漠 → 恶地

【湿度梯度（从干到湿）】
沙漠 → 稀树草原 → 平原 → 森林 → 热带雨林 → 沼泽

【典型相邻关系】
1. 沙漠常与稀树草原或恶地相邻，不会直接与雪地或热带雨林相邻。
2. 热带雨林常与沼泽或稀树草原相邻。
3. 针叶林常与雪地平原和平原之间过渡。
4. 蘑菇岛通常孤立于海洋之中，极少与其他陆地生物群系直接接壤。
5. 山地可出现在各类生物群系的边缘，作为地形隆起的变种。


# 方块与地形

## 常见地形方块

【草方块（Grass Block）】
1. 地表覆盖物，受光照和水分影响颜色。
2. 可通过在泥土上方放置透明方块让阳光照射使其生长。

【泥土（Dirt）】
1. 草方块下方数层，也是农田的基础（锄地后变为耕地）。

【石头（Stone）】
1. Y=8以下最常见的方块，挖掘可得圆石。
2. 熔炉冶炼圆石可还原为石头。

【深板岩（Deepslate）】（1.17+）
1. Y=0以下替代石头的方块，硬度更高，挖掘速度更慢。
2. 对应有深板岩版本的各类矿石。

【基岩（Bedrock）】
1. 主世界最底层（Y=-64~Y=-60）和下界顶底层，无法被生存模式破坏。

【沙子（Sand）】
1. 受重力影响会下落，主要分布在沙漠、海滩和河岸。
2. 可冶炼为玻璃。

【沙砾（Gravel）】
1. 受重力影响，主要分布于地下、山地、河床和下界。
2. 掉落物有概率为燧石（Flint）。

## 矿石分布

矿石的最佳挖掘高度（Y坐标）在1.18版本后有较大调整：

【煤炭矿石（Coal Ore）】
1. 分布：Y=0 ~ Y=320，在 Y=96 附近最为密集。
2. 用途：熔炉燃料，合成火把。

【铁矿石（Iron Ore）】
1. 分布：Y=-64 ~ Y=320，在 Y=16 和 Y=232 附近各有一个密度峰值。
2. 用途：最重要的基础金属，制作工具、防具、铁轨等。

【铜矿石（Copper Ore）】（1.17+）
1. 分布：Y=-16 ~ Y=112，在 Y=48 附近最密集，溶洞生物群系中更多。
2. 用途：铜块装饰，望远镜，避雷针。

【金矿石（Gold Ore）】
1. 分布：Y=-64 ~ Y=32，在 Y=-16 附近最密集。
2. 特殊：在恶地生物群系，金矿石在 Y=32 ~ Y=256 之间额外大量生成。
3. 用途：黄金工具（速度快但耐久低）、下界交易、金苹果。

【青金石矿石（Lapis Lazuli Ore）】
1. 分布：Y=-64 ~ Y=64，在 Y=0 附近最密集。
2. 用途：染料，附魔材料（附魔台消耗青金石）。

【红石矿石（Redstone Ore）】
1. 分布：Y=-64 ~ Y=16，越靠近基岩层越密集。
2. 用途：红石电路系统的核心材料。

【钻石矿石（Diamond Ore）】
1. 分布：Y=-64 ~ Y=16，在 Y=-58 附近最为密集。
2. 用途：最强常规工具、防具材料，也用于合成附魔台、下界合金锭。
3. 技巧：在 Y=-58 挖掘"蛇形矿道"（Branch Mining）效率最高。

【绿宝石矿石（Emerald Ore）】
1. 分布：仅在山地生物群系生成，Y=-16 ~ Y=320，Y=256 附近较多。
2. 用途：与村民交易的货币。

【下界石英矿石（Nether Quartz Ore）】
1. 分布：下界全高度范围，极为常见。
2. 用途：红石比较器，装饰方块。

【下界金矿石（Nether Gold Ore）】
1. 分布：下界全高度范围。
2. 用途：少量金粒，可用于与猪灵交易。

【远古残骸（Ancient Debris）】
1. 分布：下界 Y=8 ~ Y=119，在 Y=15 附近最密集，极为罕见。
2. 用途：冶炼后得到下界合金碎片，四个碎片合成下界合金锭，用于升级钻石装备为下界合金装备。

## 特殊结构

【村庄（Village）】
1. 生成于：平原、沙漠、稀树草原、针叶林、雪地平原等生物群系。
2. 内容：村民（可交易）、铁傀儡（守护者）、各种建筑（铁匠铺、图书馆等）、战利品箱。

【要塞（Stronghold）】
1. 生成于：地下，通过末影之眼导航找到。
2. 内容：末地传送门（需插入末影之眼激活）、图书馆、战利品箱。

【林地府邸（Woodland Mansion）】
1. 生成于：仅黑森林生物群系，距离出生点通常较远。
2. 内容：唤魔者、恼鬼、战利品箱；有时含特殊房间和图腾。

【海底神殿（Ocean Monument）】
1. 生成于：深海生物群系。
2. 内容：守卫者（Guardian）和远古守卫者（Elder Guardian）；金块核心；海绵房间。

【废弃矿井（Mineshaft）】
1. 生成于：地下，常见于峡谷壁或洞穴系统中；恶地生物群系中可能暴露在地表。
2. 内容：铁路系统、战利品箱、蜘蛛刷怪笼（带蜘蛛网）。

【沙漠庙（Desert Temple）】
1. 生成于：沙漠生物群系。
2. 内容：中央陷阱（踩压力板触发TNT爆炸），四个地下战利品箱。

【丛林庙（Jungle Temple）】
1. 生成于：热带雨林生物群系。
2. 内容：机关陷阱（绊线触发箭矢）、拼图锁谜题、战利品箱。

【古城（Ancient City）】（1.19+）
1. 生成于：仅深暗之域生物群系，极为罕见。
2. 内容：大量幽匿方块、战利品箱（含回声碎片、铁锹附魔书等稀有物品）；中心有神秘框架。
3. 注意：移动时要格外谨慎，避免触发幽匿感测体和幽匿尖啸体，否则召唤监守者。

【下界要塞（Nether Fortress）】
1. 生成于：下界，各生物群系均可出现。
2. 内容：烈焰人刷怪笼（掉落烈焰棒）、骷髅（下界骷髅）刷怪笼、下界疣、战利品箱（含稀有附魔书、鞍等）。

【猪灵堡垒（Bastion Remnant）】
1. 生成于：下界（下界荒地除外）。
2. 内容：猪灵（可交易）、猪灵蛮兵（会攻击一切）、大量战利品箱（含下界合金碎片）。


# 合成系统

## 工作台合成

合成台（Crafting Table）提供 3×3 的合成格，是游戏中最核心的合成设备。

【基础合成流程】
1. 木头（任意木材） → 木板（4个）
2. 木板（4个，放置成田字形） → 工作台
3. 木板（3个，一排）+ 木棍（2个，下方两格） → 木镐
4. 铁锭（5个，U字形）+ 木棍（2个）→ 铁桶
5. 铁锭（3个，一排）+ 木棍（2个）→ 铁剑

【常用合成配方】

【火把（Torch）】
1. 配方：木棍（下）+ 煤炭或木炭（上）
2. 产出：4个火把

【箱子（Chest）】
1. 配方：木板 8 个围成一圈（中间空）
2. 用途：储物容器；两个相邻放置合并为大箱子。

【熔炉（Furnace）】
1. 配方：圆石 8 个围成一圈（中间空）

【床（Bed）】
1. 配方：3块任意颜色羊毛（中行）+ 3块木板（下行）
2. 用途：跳过夜晚，设置重生点。

【弓（Bow）】
1. 配方：3根木棍（对角线排列）+ 3根线（另一侧对角线）

【附魔台（Enchanting Table）】
1. 配方：书（中上）+ 钻石（两侧中行）+ 黑曜石（下行三块 + 中间一块）

【末影之眼（Eye of Ender）】
1. 配方：末影珍珠 + 烈焰粉

【下界合金锭（Netherite Ingot）】
1. 配方：4个下界合金碎片 + 4个金锭（围绕中心）

## 熔炉冶炼

熔炉（Furnace）使用燃料将原材料加工为产品。

【常见冶炼配方】
1. 铁矿石 / 铜矿石 / 金矿石 → 对应金属锭
2. 圆石 → 石头
3. 沙子 → 玻璃
4. 黏土块 → 砖块
5. 木头 → 木炭（替代煤炭的燃料和火把材料）
6. 陶土 → 烧制陶土（Terracotta）
7. 下界合金碎片（Ancient Debris 的冶炼产物）→ 原材料

【燃料效率（从低到高）】
1. 树苗/木板：燃烧时间约1.5格/个，效率低但来源丰富。
2. 煤炭/木炭：燃烧时间8格/个，最常用的燃料。
3. 煤炭块：燃烧时间80格/个（9个煤炭合成），效率最高的常规燃料。
4. 岩浆桶：燃烧时间100格，且桶可回收再用，为最高效燃料之一。

## 其他加工设备

【锻造台（Smithing Table）】（1.20重做）
1. 用途：升级钻石装备为下界合金装备，也可更换装备款式。
2. 配方：下界合金模板 + 钻石装备 + 下界合金锭

【砂轮（Grindstone）】
1. 用途：合并同类工具修复耐久，或去除附魔（不能去除诅咒）。

【铁砧（Anvil）】
1. 用途：修复装备（保留附魔）、重命名物品、将附魔书上的附魔附加到装备上。

【织布机（Loom）】
1. 用途：制作旗帜图案，操作比工作台更直观。

【制图台（Cartography Table）】
1. 用途：复制、缩放或锁定地图。

【切石机（Stonecutter）】
1. 用途：将石类方块切割为各种变体（台阶、楼梯、墙等），比工作台合成更高效。


# 工具与武器

## 工具等级

工具等级决定了可以挖掘的方块类型和速度，从低到高分为：
木制 < 石制 < 铁制 < 金制 < 钻石制 < 下界合金

【注意】金制工具挖掘速度最快但耐久极低；下界合金工具速度快、耐久高，且不会被熔岩/火焰损坏。

## 常用工具

【镐（Pickaxe）】
1. 用途：挖掘石类方块和矿石，是游戏中最重要的工具。
2. 注意：铁矿石及以上的矿石必须用石制或以上的镐才能获得掉落物；钻石、黑曜石需铁制及以上；远古残骸需钻石或下界合金镐。

【斧（Axe）】
1. 用途：快速砍伐木类方块，也可作为武器（伤害较高但攻速慢）。
2. 技巧：在 Java 版中，斧攻击会取消盾牌防御约5秒。

【锹（Shovel）】
1. 用途：挖掘泥土、沙子、沙砾、雪等软性方块；右键可将泥土路径化。

【锄（Hoe）】
1. 用途：耕地用于种植，1.17 版本后可快速挖掘叶子、草地等自然方块。

【剪刀（Shears）】
1. 用途：剪羊毛（不损坏羊）、收集树叶（不掉落）、剪蜘蛛网。

【火石与打火石（Flint and Steel）】
1. 用途：点火（激活下界传送门、点燃TNT）。

## 武器

【剑（Sword）】
1. 攻击速度快，伤害稳定，是最主流的近战武器。
2. 可附魔：锋利、横扫之刃（Java版）、火焰附加、击退、掠夺等。

【弓（Bow）】
1. 远程武器，需要箭矢，充能满后伤害最高。
2. 可附魔：力量、火焰、无限（存一支箭即可无限射击）、穿透。

【弩（Crossbow）】（1.14+）
1. 可提前装填，射击速度快于弓，但充能时间较长。
2. 可附魔：穿刺（射穿多个目标）、多重射击（同时发射3支箭）、快速装填。

【三叉戟（Trident）】
1. 水下战斗的最佳武器，既可近战又可投掷。
2. 只能从溺尸身上掉落获取，无法合成。
3. 可附魔：激流（水中冲刺）、引雷（吸引闪电）、忠诚（投出后自动飞回）。


# 防具系统

防具分为头盔、胸甲、护腿、靴子四个部位，材质等级与工具相同：皮革 < 锁链 < 铁制 < 金制 < 钻石制 < 下界合金。

【防具属性】
1. 防护值：头盔3、胸甲8、护腿6、靴子3（铁制满套共20点）。
2. 耐久：下界合金最高，皮革最低。
3. 特殊：下界合金防具不会被熔岩/火焰损坏；锁链甲只能通过战利品获得，无法合成。

【常用防具附魔】
1. 保护（Protection）：减少各类伤害，最通用。
2. 火焰保护（Fire Protection）：减少火焰伤害。
3. 爆炸保护（Blast Protection）：减少爆炸伤害（尤其对抗苦力怕）。
4. 摔落保护（Feather Falling）：减少摔落伤害，靴子专用。
5. 荆棘（Thorns）：反弹攻击者部分伤害，但会加速防具损耗。
6. 深海探索者（Depth Strider）：水中行走速度提升，靴子专用。
7. 冰霜行者（Frost Walker）：走过水面时冻成冰，靴子专用。
8. 灵魂疾行（Soul Speed）：在灵魂沙和灵魂土上加速，靴子专用。

【盾牌（Shield）】
1. 右键举起可抵挡正面伤害（包括箭矢）。
2. Java版中，斧攻击可以使盾牌暂时失效约5秒。
3. 可在铁砧中添加旗帜图案。


# 生物（Mob）

## 被动生物

【猪（Pig）】
1. 掉落：生猪肉（死亡时如身上有火则掉落熟猪肉）。
2. 繁殖：胡萝卜、甜菜根、马铃薯。
3. 特殊：可装上鞍后骑乘，用胡萝卜钓竿控制方向。

【牛（Cow）】
1. 掉落：牛肉、皮革。
2. 繁殖：小麦。
3. 特殊：可用桶挤牛奶（桶装牛奶可清除所有状态效果）。

【羊（Sheep）】
1. 掉落：羊毛、羊肉。
2. 繁殖：小麦。
3. 特殊：可用剪刀剪毛（不杀死），用染料右键可改变羊毛颜色，生成粉红羊的概率极低（~0.2%）。

【鸡（Chicken）】
1. 掉落：生鸡肉、羽毛；定时产鸡蛋（可投掷，有概率孵出小鸡）。
2. 繁殖：任意种子。

【马（Horse）】
1. 繁殖/驯服：小麦、苹果、糖、面包、金萝卜等食物多次喂食（有心形出现则驯服成功）。
2. 特殊：装上鞍可骑乘，装上马铠可提高防御；速度和跳跃高度随机，好马极为珍贵。

【村民（Villager）】
1. 各职业村民提供不同交易内容（图书管理员可交易附魔书）。
2. 可通过喂食繁殖，需要床和食物充足。
3. 僵尸化后可用金苹果+虚弱药水治愈，治愈后交易价格永久下降。

## 中立生物

【蜜蜂（Bee）】
1. 在蜂巢周围活动，授粉后回巢。
2. 攻击时会给玩家中毒，随后蜜蜂死亡。
3. 用剪刀采蜜巢可获得蜂蜜瓶（不激怒蜜蜂需在蜂巢下放营火）。

【猪灵（Piglin）】
1. 生存于下界，对穿着金甲的玩家友好，可进行黄金交易。
2. 扔给猪灵金锭，随机掉落下界物品（包括下界合金碎片！）。
3. 带入主世界会僵尸化，变为僵尸猪灵。

【末影人（Enderman）】
1. 正视会触发攻击，可通过戴南瓜头盔避免激怒。
2. 掉落末影珍珠（传送必备材料、合成末影之眼）。
3. 在水中受伤，会自动传送逃离。

【蜘蛛（Spider）】
1. 白天（亮度足够）中立，夜晚变为敌对。
2. 掉落蜘蛛眼和线，线可合成弓、钓鱼竿等。

【狼（Wolf）】
1. 用骨头驯服后成为狗，可跟随主人战斗。
2. 受伤时可喂食肉类回血；繁殖用骨头（驯服后）。

## 敌对生物

【苦力怕（Creeper）】
1. 靠近玩家后嘶嘶作响并爆炸，是玩家最大的威胁之一。
2. 被闪电击中变为充能苦力怕，爆炸威力大幅提升。
3. 掉落火药（制作TNT、烟花、喷溅药水）；被骷髅的箭矢杀死时掉落唱片。
4. 对策：保持距离用弓射击；近战用剑击退后立刻后退。

【骷髅（Skeleton）】
1. 远程弓箭攻击，精准且射速快。
2. 掉落骨粉（骨头合成）和箭矢，偶尔掉落弓。
3. 骨粉可用于快速生长作物。
4. 对策：格挡后近战，或用盾牌格挡箭矢。

【僵尸（Zombie）】
1. 近战攻击，可击破木门（普通难度及以上）。
2. 掉落腐肉（可食用但会造成食物中毒，饥饿值低时也可用）、铁锭（低概率）。
3. 阳光下会燃烧。

【溺尸（Drowned）】
1. 水中的僵尸变种，会用三叉戟攻击。
2. 唯一掉落三叉戟的生物（概率极低）；也掉落铜锭（1.17+）。

【蠹虫（Silverfish）】
1. 藏身于石头、圆石、石砖中，破坏这些方块时有概率触发。
2. 要塞中大量存在，需小心。

【幻翼（Phantom）】
1. 玩家连续超过3个游戏日不睡觉时，夜晚开始生成并俯冲攻击。
2. 掉落幻翼膜（合成缓降药水的材料）。
3. 对策：睡觉或猫会驱赶幻翼。

【女巫（Witch）】
1. 使用各种药水攻击和防御，抗性较高。
2. 掉落各种药水材料，常见于沼泽小屋。

【监守者（Warden）】（1.19+）
1. 深暗之域的守护者，没有视觉，靠振动和气味感知玩家。
2. 血量极高（500点），伤害极大，是游戏中最强的生物之一。
3. 三次触发幽匿尖啸体后召唤，无掉落物，逃跑即可。
4. 对策：蹲伏缓慢移动，避免激活幽匿感测体；一旦遇见直接跑路。

## BOSS

【末影龙（Ender Dragon）】
1. 生成于：末地中央岛。
2. 攻略：用弓射击末地水晶（末端方尖塔上）切断治疗来源，然后等末影龙俯冲时近战攻击头部（对头部伤害最高）。
3. 掉落：大量经验、末地传送门（回归主世界）、龙蛋（装饰品）。
4. 可重召唤：用4个末地水晶放置于末地传送门四侧即可重召唤，每次击杀获得更少经验。

【凋灵（Wither）】
1. 召唤方式：用3个凋灵骷髅头 + 4个灵魂沙/灵魂土拼成T字形。
2. 攻略：护甲齐全、喝下抗性提升和生命恢复药水，远程弓箭配合近战。初期生成时有护盾，等护盾消失后全力攻击。
3. 掉落：凋灵之星（合成信标的核心材料）。
4. 建议在下界或地下空间召唤，避免破坏地表建筑。

【远古守卫者（Elder Guardian）】
1. 生成于：海底神殿，每座有3只。
2. 攻击方式：给予挖掘疲劳效果，使玩家无法有效挖掘；激光攻击伤害高。
3. 掉落：海绵、鱼类，偶尔掉落金锭。
4. 对策：喝下牛奶消除挖掘疲劳，携带大量食物和高等装备。


# 农业与食物

## 基础农业

【农田建立】
1. 靠近水源（5格以内）的泥土用锄耕地变为农田（Farmland）。
2. 骨粉可立即使作物成熟（部分作物）。
3. 需要光照等级9以上才能生长。

【常见作物】
1. 小麦：最基础的作物，掉落小麦和种子。可合成面包、蛋糕。
2. 胡萝卜：收获即为种子和食物。生吃或合成金胡萝卜（耐饱值高）。
3. 马铃薯：收获即为食物，可烤制为烤马铃薯（更高饱食度）。
4. 甜菜根：掉落甜菜根（制汤）和甜菜根种子；也是红色染料来源。
5. 西瓜：需要在旁边一格生长瓜，右键可分离1~9片西瓜片。
6. 南瓜：类似西瓜，可戴在头上（防止末影人攻击），也用于合成铁/雪/南瓜灯。
7. 甘蔗：水边自然生长，可高达3格，用于合成纸（书籍、地图）和糖。
8. 竹子：热带雨林中生长，生长极快，是熊猫的食物和脚手架的材料。

## 食物与饥饿系统

玩家有10格（20点）饥饿值，饱食度影响回血速度：
1. 饥饿值满时，生命值每4秒恢复1点（普通难度）。
2. 饥饿值降至0时，开始扣血（极限难度可致死）。
3. 疾跑、跳跃、挖掘等行为额外消耗饥饿值。

【高效食物推荐】
1. 金胡萝卜：饱食度19.2，最高效的单格饱食度食物之一。
2. 烤猪肉/牛排：饱食度12.8，容易大量获取。
3. 面包：饱食度7.6，小麦易于批量种植。
4. 金苹果：额外提供生命恢复和吸收效果，用于紧急恢复或治疗僵尸村民。
5. 附魔金苹果（附魔金苹果/Enchanted Golden Apple）：提供强力增益效果，极为珍贵，无法合成，只能在战利品箱中找到。


# 红石系统

红石（Redstone）是 Minecraft 的电路系统，可以制造复杂的自动化机器。

【基本元件】
1. 红石粉：传导红石信号，最长15格，每格信号减弱1。
2. 红石火把：持续发出信号，也可作为信号反相器（NOT门）。
3. 红石中继器：单向传导信号，可延时0.1~0.4秒，也可锁定信号。
4. 红石比较器：比较两路信号强度，或检测容器内容物（信号强度对应物品数量）。
5. 红石灯：接收信号时发光。

【常用输入元件】
1. 按钮：短暂触发信号（木制1秒，石制0.5秒）。
2. 拉杆：持续开关信号。
3. 压力板：玩家/生物走上时触发；金制/铁制只对特定对象响应。
4. 绊线钩：线被触碰时触发。
5. 阳光感测器：根据天空光照强度输出信号，白天满信号。
6. 观察者：检测相邻方块状态变化，输出短暂信号。

【常用输出元件】
1. 活塞：推动方块（最多12格）；粘性活塞可拉回方块。
2. 发射器：向前方发射物品（箭矢、火球、喷溅药水等）。
3. 投掷器：向前方丢出物品（不激活物品效果）。
4. 漏斗：自动传输物品到容器，或从容器抽取物品。
5. 铁门/铁活板门：需红石信号才能开关（不能手动开关）。

【实用装置】
1. 自动农场：利用活塞推动成熟作物，漏斗收集。
2. 自动分类器：利用比较器检测容器信号，漏斗传输物品。
3. 时钟电路：利用中继器循环产生周期性信号。
4. TNT大炮：利用TNT爆炸推动另一TNT到远处爆炸。


# 附魔系统

附魔（Enchanting）可以为工具、武器和防具添加特殊效果。

【附魔台（Enchanting Table）】
1. 消耗：青金石 + 经验等级。
2. 等级与书架数量：附魔台周围放置书架（最多15个，距离2格以内，中间不能有方块遮挡）可提高附魔等级，最高提供30级附魔选项。
3. 每次可选3个附魔选项，附魔等级越高，获得强力附魔的概率越大。

【铁砧附魔书】
1. 将附魔书与装备在铁砧合并，可将书上的附魔转移到装备上。
2. 每次铁砧操作会增加"修复费用"，费用过高时显示"太贵了！"（费用超过39级）。

【常见附魔】

【工具附魔】
1. 效率（Efficiency）：提高挖掘速度，最高V级。
2. 精准采集（Silk Touch）：挖掘方块时获得方块本身（如冰块、草方块、矿石原矿）。
3. 时运（Fortune）：增加矿石掉落数量（钻石、煤炭、青金石等），最高III级。
4. 耐久（Unbreaking）：降低装备耐久损耗概率，最高III级。
5. 修补（Mending）：获得经验时自动修复手持或穿戴的装备，极为珍贵。

【冲突附魔】精准采集与时运不能同时存在于同一工具上。

【武器附魔】
1. 锋利（Sharpness）：增加攻击伤害，最高V级。
2. 亡灵克星（Smite）：对亡灵生物额外伤害，最高V级。
3. 节肢克星（Bane of Arthropods）：对蜘蛛类生物额外伤害，最高V级。
4. 掠夺（Looting）：增加怪物掉落物数量和概率，最高III级。
5. 火焰附加（Fire Aspect）：击中时点燃目标，最高II级。
6. 击退（Knockback）：增加击退距离，最高II级。

【防具附魔】
1. 保护（Protection）：综合减伤，最高IV级。
2. 摔落保护（Feather Falling）：靴子专用，减少摔落伤害。
3. 荆棘（Thorns）：反弹伤害，最高III级。
4. 修补（Mending）：同工具，极为实用。

【诅咒附魔（负面）】
1. 消失诅咒（Curse of Vanishing）：死亡时装备直接消失。
2. 束缚诅咒（Curse of Binding）：无法卸下被附魔的盔甲。


# 进度与成就系统

Minecraft（Java版）使用进度（Advancements）系统来引导玩家逐步深入游戏，基岩版使用成就（Achievements）系统。

【主要进度线索】

【Minecraft（起点）】
1. 获得木板 → 拿到工作台 → 制作工具 → 进入地下 → 获得矿石

【下界进度线】
1. 制作下界传送门 → 进入下界 → 找到下界要塞 → 获取烈焰棒 → 酿造药水 → 击败凋灵

【末地进度线】
1. 合成末影之眼 → 找到要塞 → 激活传送门 → 进入末地 → 击败末影龙

【冒险进度线（Adventure）】
1. 与村民交易 → 发现各类结构 → 击败各类BOSS

【Nether进度（We Need to Go Deeper）】
1. 进入下界的第一步。


# Tips - 玩家可能不知道的知识

- 手持食物右键村民可以繁殖村民，无需任何额外条件，但村民需要有足够的床位。
- 南瓜头盔可以防止末影人主动攻击你（直视不触发敌对），这对在末地行动非常有用。
- 下界传送门在下界中对应坐标需要除以8（主世界X/Z ÷ 8 = 下界对应坐标），充分利用这一点可以建立高效传送网络。
- 床在下界和末地使用会爆炸，爆炸威力远大于TNT，可以用于挖矿（尤其是下界挖远古残骸）但要注意防止自伤。
- 苦力怕害怕猫，在基地附近养猫可以有效减少苦力怕靠近。骷髅害怕狼，猪灵害怕猪灵蛮兵头颅。
- 末影人不会主动攻击戴着南瓜头的玩家，在末地或主世界收集末影珍珠时非常有用。
- 红石中继器锁定功能（侧面输入信号）可以用于制作内存和锁存器，是复杂红石电路的基础。
- 用丝绸之触镐挖掘刷怪笼不会将其破坏（1.20前无法获取），但可以用工作台或砂轮重置其状态。（1.20.2+可用精准采集获取刷怪笼）
- 村民被僵尸化后用虚弱药水（投掷型）和金苹果治愈，该村民的所有交易价格会永久降为最低，是获取廉价附魔书的重要途径。
- 钓鱼不仅可以钓鱼，还有概率获得附魔书、附魔渔竿、马鞍、弓等宝藏物品；使用时运III + 海之眷顾III的附魔渔竿效率最高。
- 女巫在受到攻击时会自动喝下药水进行防御（如抗火、速度药水），所以直接击杀比让她一直喝药效率更高。
- 末影龙对爆炸有免疫，TNT对其无效，但对末地水晶有效（注意爆炸波及范围）。
- 铁傀儡可以由玩家用4个铁块（T字形）+ 1个南瓜头合成，是保护村庄的利器，也是铁农场的核心。
- 凋零在血量降至一半时会产生护盾，此时只有近战能伤害它。可以在狭窄隧道中召唤凋零，利用天花板限制其移动。
- 海豚（Dolphin）跟随玩家游泳时会给予"海豚的恩惠"效果，大幅提升水中移动速度，是探索海洋时的好伙伴。
- 下界岩浆中使用火焰保护IV套装+抗火药水可以短暂在岩浆中游泳，用于紧急情况下逃生或探索。
- 竹子是游戏中生长最快的植物，且可以堆肥获得骨粉；利用观察者+活塞可以制作高效的竹子自动收割农场。
- 闪烁的西瓜片（Glistering Melon）无法食用，只用于合成金胡萝卜的替代品——不，它用于酿造治疗药水。
- 在1.18+版本中，Y=-58左右挖矿是获取钻石最高效的高度，同时配合支洞挖掘（每两格一条）覆盖率最高。
- 猪灵（Piglin）交易时扔金锭后会有随机等待时间，交易物品池包括下界合金碎片（概率约1.8%），大量金锭交易是早期获取下界合金的方法之一。
- 监守者（Warden）无法被击杀获得战利品（无掉落物），游戏设计意图是让玩家逃跑而非战斗。但如果你真的想击杀，需要极强的装备和药水支持。
