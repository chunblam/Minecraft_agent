## 酿造药水
1. 将1-3个水瓶或药水放入酿造台界面底部的药水槽中，将材料放入顶部的材料槽中，再在燃料槽里放置烈焰粉，便可开始酿造。
2. 药水的酿造一般从水瓶开始，水瓶可以用 [玻璃瓶] 从 [水源] 或 [炼药锅] 里盛水获得。在酿造台具有能量的前提下，添加基础材料来制作基础药水，这一步的产物大部分不具有效果（一般情况下，需要在这一步使用 [下界疣] 制作 [粗制的药水]）。在此基础上，继续放入影响材料，可以制作出效果药水。继续向具有效果的药水中添加改性剂，可以使药水效果增强、变得更加持久或是腐化现有的效果。
3. 加入 [火药]，可以将药水转化成喷溅药水，使药水可以投掷（或被 [发射器] 发射），造成范围效果。还可以对喷溅药水加入 [龙息]，将药水转化成滞留药水，使药水投掷后产生一片 [区域效果云]。
4. 每次酿造耗时总是20秒，与材料种类无关。
5. 酿造需要能量才能进行，在燃料槽放入 [烈焰粉] 可以为酿造台增加能量，每份烈焰粉使酿造台增加20点能量，即可以进行20次酿造。在 [Java版] 中，酿造需要消耗1点能量才能开始，材料被取出等原因导致酿造终止时，消耗的能量不会返还。在 [基岩版] 中，只有酿造过程结束之后才会消耗能量，中途取出酿造材料不会损失能量。

## 酿造设备
酿造系统核心物品及用途，以下是用于酿造系统的关键物品及其主要功能。
【酿造台】
用途： 酿造设备的核心。用于向水瓶或药水中添加各种材料，以酿造出具有不同效果的药水。
【炼药锅】
用途： 主要用于储存水。玩家可以用玻璃瓶从炼药锅中装水，一个装满水的炼药锅可以填满3个玻璃瓶。
版本差异： 仅在基岩版中，炼药锅还可以用于储存药水。
【烈焰粉】
用途： 作为酿造台的燃料。每次酿造操作需要消耗1个烈焰粉。
【玻璃瓶】
用途： 作为所有药水的基础容器。空玻璃瓶是制作和盛装药水的必备物品。
【水瓶】
用途： 是所有类型药水的基础原料。任何药水的酿造都必须从水瓶开始。
获取方式： 手持玻璃瓶对水源方块或装有水的炼药锅使用即可获得。
【总结说明】
工作流程： 酿造的基本流程是：用玻璃瓶从水源或炼药锅装水获得水瓶，然后在酿造台中，以烈焰粉为燃料，向水瓶中添加材料来制作药水。
核心角色： 酿造台是加工中心，烈焰粉是能源，玻璃瓶和水瓶是基础容器与原料，炼药锅则是一个便捷的储水工具。

## 材料
### 【基础材料与改性剂】
基础材料是直接加入水瓶的材料，是所有药水的起点。 [下界疣] 是此阶段的最基本材料，需要用它来制作绝大多数的药水。
改性剂是用来改变药水属性或是使药水效果相反的材料。发酵蛛眼是唯一可以直接酿造成有效药水的改性剂。
以下是用于酿造药水的基础材料和能够改变药水性质的改性剂。
【下界疣】
1. 加入水瓶时得到： 粗制的药水
2. 改性剂效果： 无。它是酿造几乎所有功能药水必需的基础材料，本身不改变药水效果。
【荧石粉】
1. 加入水瓶时得到： 浓稠的药水
2. 改性剂效果： 提升效果等级。例如，将“再生”药水变为“再生 II”药水。
【红石粉】
1. 加入水瓶时得到： 平凡的药水
2. 改性剂效果： 延长效果持续时间。例如，将持续3分钟的夜视药水延长至8分钟。
【发酵蛛眼】
1. 加入水瓶时得到： 虚弱药水
2. 改性剂效果： 腐化效果。它会将原有药水效果转变为另一种通常具有负面或相反效果的药水（具体转换关系见下文详述）。
【火药】
1. 加入水瓶时得到： 喷溅药水
2. 改性剂效果： 将任何普通药水变为喷溅药水，使其可以被投掷，对他人或自己（通过撞击）生效。
【龙息】
1. 加入水瓶时得到： 滞留药水（当加入喷溅药水时）
2. 改性剂效果： 将任何喷溅药水变为滞留药水。投掷后会在落点形成一片滞留效果云，使经过的生物持续受到药水效果影响。
【总结说明】
酿造流程： 标准酿造流程通常始于用下界疣将水瓶升级为“粗制的药水”，然后加入效果材料（如烈焰粉制成迅捷药水），最后根据需要，使用荧石粉（强化）、红石粉（延长）或发酵蛛眼（腐化）进行改性。
形态转变： 火药和龙息不改变药水效果本身，而是改变其使用形态，分别用于制作可投掷的喷溅药水和范围持续的滞留药水。
【**腐化效果**】：发酵蛛眼会改变药水的基本效果，通常会产生与原效果相反的负面效果。
1. 腐化药水通常与原药水效果相反。
2. 腐化支持相同改性剂的延长版或增强版的药水会得到带有相同改性效果的药水。
3. 腐化延长版的剧毒药水只会让其变成普通版的伤害药水，增强版的跳跃药水和迅捷药水不能腐化。
4. 隐身药水是夜视药水的腐化版本。也就是说，它将夜视药水“外界于你的可视度变到最高”的效果反转成“你于外界的可见度变为零”，即：隐身。
【**喷溅**】和 【**滞留**】药水：任何药水都可以变为喷溅药水或滞留药水，后者是长时间滞留的药水。
1. 喷溅药水和滞留药水都可以像它们的普通药水一样改性。例如，喷溅的水瓶可以加入任何基础材料，得到的依然是喷溅药水。
2. 喷溅药水和可饮用版本的持续时间相同。
3. 滞留药水的持续时间只有可饮用版本的四分之一。例如，8:00的可饮用药水变为滞留药水后只有2:00。

### 影响材料
影响材料将给予粗制的药水一个特定效果，但不会改变药水强度和时间。
若直接加入到水瓶中，任何这些材料都只会造出无效果的药水（ [河豚]、 [金胡萝卜]、 [海龟壳]、 [幻翼膜] 除外，它们不能加入到水瓶中来制造药水）。
以下材料可用于为“粗制的药水”添加特定的状态效果。当这些材料与“发酵蛛眼”结合时，会产生所谓的“腐化效果”，将原效果转变为另一种效果。

1. 效果： 迅捷
2. 腐化效果： 缓慢
【兔子脚】
1. 效果： 跳跃提升
2. 腐化效果： 无
【闪烁的西瓜片】
1. 效果： 瞬间治疗
2. 腐化效果： 瞬间伤害
【蜘蛛眼】
1. 效果： 中毒
2. 腐化效果： 无
【河豚】
1. 效果： 水下呼吸
2. 腐化效果： 无
【岩浆膏】
1. 效果： 抗火
2. 腐化效果： 无
【金胡萝卜】
1. 效果： 夜视
2. 腐化效果： 隐身
【烈焰粉】
1. 效果： 力量
2. 腐化效果： 无
【恶魂之泪】
1. 效果： 生命恢复
2. 腐化效果： 无
【海龟壳】
1. 效果： 神龟之力（同时提供抗性提升和缓慢效果）
2. 腐化效果： 无
【幻翼膜】
1. 效果： 缓降
2. 腐化效果： 无
【石头】
1. 效果： 寄生 (Minecraft: Dungeons 中的效果，非原版Minecraft药水效果)
2. 腐化效果： 无
【黏液块】
1. 效果： 渗浆 (Minecraft: Dungeons 中的效果，非原版Minecraft药水效果)
2. 腐化效果： 无
【蜘蛛网】
1. 效果： 盘丝 (Minecraft: Dungeons 中的效果，非原版Minecraft药水效果)
2. 腐化效果： 无
【旋风棒】
1. 效果： 蓄风 (Minecraft: Dungeons 中的效果，非原版Minecraft药水效果)
2. 腐化效果： 无
【总结说明】：
标准药水效果： 大部分材料用于酿造原版Minecraft中标准的正面或负面药水。
腐化机制： 仅有少数几种药水效果可以通过加入发酵蛛眼被“腐化”转变为另一种效果，例如：
迅捷 ↔ 缓慢
瞬间治疗 ↔ 瞬间伤害
夜视 ↔ 隐身
特殊说明： 表格末尾列出的几种材料（石头、黏液块、蜘蛛网、旋风棒）所对应的效果（寄生、渗浆、盘丝、蓄风）并非原版Minecraft的酿造内容，而是来自《Minecraft: Dungeons》游戏。在原版Minecraft中使用这些材料进行酿造是无效的。


## 酿造配方
### 【基础药水】
基础药水是没有效果的药水。水瓶以外的所有基础药水都是在水瓶里加入单个材料制作而来的。
以下配方均遵循标准的酿造台布局：将基础材料放入上方单个槽位，将水瓶或前置药水放入中间下方的三个槽位中进行酿造。
【水瓶】
1. 用途：作为所有药水酿造的基础原料。
2. 用于酿造：粗制的药水、平凡的药水、浓稠的药水、虚弱药水。
【粗制的药水】
1. 配方：
- 上方槽位： 下界疣
- 下方槽位： 水瓶
2. 用途： 酿造所有效果药水的必备前置药水。
【平凡的药水】
1. 配方：
- 上方槽位： 红石粉、糖、兔子脚、闪烁的西瓜片、蜘蛛眼、岩浆膏、烈焰粉、恶魂之泪、石头、黏液块、蜘蛛网、旋风棒 等多种材料之一
- 下方槽位： 水瓶
2. 用途：
- Java版： 无直接用途。
- 基岩版： 可用于酿造虚弱药水。
【浓稠的药水】
1. 配方：
- 上方槽位： 荧石粉
- 下方槽位： 水瓶
2. 用途：
- Java版： 无直接用途。
- 基岩版： 可用于酿造虚弱药水。
【总结说明】：
酿造起点： 水瓶是所有药水的起点。
核心步骤： 使用下界疣将水瓶酿造为粗制的药水是制作绝大多数功能药水的关键且必需的步骤。
版本差异： 平凡的药水和浓稠的药水在Java版中属于中间产物，没有进一步的酿造用途；而在基岩版中，它们都可以作为酿造虚弱药水的原料。

### 效果药水
效果药水通过将单个材料加入粗制的药水酿造而来（虚弱药水除外），可以和 [火药] 结合以制造其对应的喷溅药水。然而，某些药水需要使用 [发酵蛛眼] 腐化其他药水来得到。
#### 【正面效果】
以下配方均遵循标准的酿造台布局：将基础材料放入上方单个槽位，将水瓶或前置药水放入中间下方的三个槽位中进行酿造。
下表详细列出了各种具有正面效果的药水及其【基础配方】、【延长版本】和【增强版本】的酿造方法与效果。
【治疗药水】
1. 基础配方： 闪烁的西瓜片 + 粗制的药水
- 延长版： 无
- 增强版： 荧石粉 + 治疗药水
2. 效果：
- 基础版： 瞬间治疗 I，立即恢复4点（）生命值。
- 增强版： 瞬间治疗 II，立即恢复8点（）生命值。
【抗火药水】

2. 效果： 抗火，获得对火、熔岩、烈焰人火球冲击与岩浆块、营火效果的免疫。

2. 效果：
- 基础版： 生命恢复 I，每2.5秒恢复1点（）生命值。
- 增强版： 生命恢复 II，每1.25秒恢复1点（）生命值。
【力量药水】
1. 基础配方： 烈焰粉 + 粗制的药水 (3:00)
- 延长版： 红石粉 + 力量药水 (8:00)
- 增强版： 荧石粉 + 力量药水 (1:30)
2. 效果：
- 基础版： 力量 I，增加3点（）近战伤害[Java版] 或 增加30%近战攻击伤害+1点（）[基岩版]。
- 增强版： 力量 II，增加6点（）近战伤害[Java版] 或 增加69%近战攻击伤害+2.3点（）[基岩版]。
【迅捷药水】
1. 基础配方： 糖 + 粗制的药水 (3:00)
- 延长版： 红石粉 + 迅捷药水 (8:00)
- 增强版： 荧石粉 + 迅捷药水 (1:30)
2. 效果：
- 基础版： 迅捷 I，提升约20%的移动速度、疾跑速度与跳跃距离。
- 增强版： 迅捷 II，提升约40%的移动速度、疾跑速度与跳跃距离。
【夜视药水】
1. 基础配方： 金胡萝卜 + 粗制的药水 (3:00)
- 延长版： 红石粉 + 夜视药水 (8:00)
- 增强版： 无
2. 效果： 夜视，使一切区域以最大亮度显示，包括水下区域。
【隐身药水】
1. 基础配方： 发酵蛛眼 + 夜视药水 (3:00)
- 延长版： 红石粉 + 隐身药水 (8:00)
- 增强版： 无
2. 效果： 隐身，将实体渲染为透明的，但装备着或手持的物品仍然可见。
【水肺药水】
1. 基础配方： 河豚 + 粗制的药水 (3:00)
- 延长版： 红石粉 + 水肺药水 (8:00)
- 增强版： 无
2. 效果： 水下呼吸，在水下时氧气条不会耗尽，并会缓慢恢复氧气条。
【跳跃药水】
1. 基础配方： 兔子脚 + 粗制的药水 (3:00)
- 延长版： 红石粉 + 跳跃药水 (8:00)
- 增强版： 荧石粉 + 跳跃药水 (1:30)
2. 效果：
- 基础版： 跳跃提升 I，将跳跃高度提升¼个方块，并减少1点（）摔落伤害。
- 增强版： 跳跃提升 II，将跳跃高度提升1½个方块，并减少2点（）摔落伤害。
【缓降药水】
1. 基础配方： 幻翼膜 + 粗制的药水 (1:30)
- 延长版： 红石粉 + 缓降药水 (4:00)
- 增强版： 无
2. 效果： 缓降，显著减缓降落速度，并在着地时完全免疫摔落伤害。
【总结说明】：
通用酿造规则：
所有基础效果药水（除隐身药水外）均由粗制的药水与对应的效果材料酿造。
延长版通常通过向基础药水添加红石粉来制作，大幅增加效果持续时间。
增强版通常通过向基础药水添加荧石粉来制作，提升效果等级但会缩短持续时间。
特殊配方： 隐身药水是唯一一个需要使用另一成品药水（夜视药水）作为基础，并通过发酵蛛眼进行“腐化”来制作的药水。

#### 负面效果 
以下配方均遵循标准的酿造台布局：将基础材料放入上方单个槽位，将水瓶或前置药水放入中间下方的三个槽位中进行酿造。
下表详细列出了具有负面效果或来自衍生游戏的药水配方与效果。
【剧毒药水】
1. 基础配方： 蜘蛛眼 + 粗制的药水 (0:45)

#### 混合效果 
以下配方均遵循标准的酿造台布局：将基础材料放入上方单个槽位，将水瓶或前置药水放入中间下方的三个槽位中进行酿造。
【神龟药水】是一种提供强大防御但会牺牲移动能力的特殊药水。
【神龟药水】
1. 基础配方： 海龟壳 + 粗制的药水 (0:20)
- 延长版： 红石粉 + 神龟药水 (0:40)
- 增强版： 荧石粉 + 神龟药水 (0:20)
2. 效果：
- 基础版与延长版： 同时提供 【缓慢 IV】 和 【抗性提升 III】 效果。
【缓慢 IV】 将移动速度降低至正常速度的约40%。
【抗性提升 III】 将所受的所有伤害减少60%（即只承受40%的伤害）。
- 增强版： 效果提升为 【缓慢 VI】 和 【抗性提升 IV】。
【缓慢 VI】 将移动速度大幅降低至正常速度的约10%。
【抗性提升 IV】 将所受的所有伤害减少80%（即只承受20%的伤害）。
【总结说明】：
双重效果： 神龟药水是唯一一种能同时提供缓慢和抗性提升两种效果的药水，完美体现了“龟壳”的防御与笨重特性。
效果平衡： 该药水通过牺牲机动性来换取极高的伤害减免，适合在需要硬扛伤害的场合使用。
酿造路径：
使用红石粉可以延长基础效果的持续时间，从20秒延长至40秒。
使用荧石粉可以增强效果等级，获得更高的伤害抗性和更慢的移动速度，但持续时间保持不变（20秒）。

### 药物 
**此段落描述的是教育版相关特性。**
该特性仅在 [教育版] 和开启了“Education Edition”选项的 [基岩版] 世界中可用。
[药物] 可由粗制的药水加入不同的 [元素] 酿造成，使用后可以去除特定的效果。药物不是药水，因此没有喷溅、滞留、增强、延长的版本。
以下配方均遵循标准的酿造台布局：将基础材料放入上方单个槽位，将水瓶或前置药水放入中间下方的三个槽位中进行酿造。
药物是一种特殊的酿造产物，它由粗制的药水加入不同的元素制成，其作用是移除玩家身上的特定负面状态效果。药物在分类上不属于药水，因此它没有喷溅型、滞留型、增强型或延长型等变种。
【解毒剂】
1. 配方： 银 + 粗制的药水
2. 效果： 饮用后立即去除中毒效果。
【灵药】
1. 配方： 钴 + 粗制的药水
2. 效果： 饮用后立即去除虚弱效果。
【眼药水】
1. 配方： 钙 + 粗制的药水
2. 效果： 饮用后立即去除失明效果。
【补药】
1. 配方： 铋 + 粗制的药水
2. 效果： 饮用后立即去除反胃效果。
【总结说明】：
核心功能：所有药物的核心功能都是清除特定的负面状态效果，而非像药水那样施加效果。
酿造基础：所有药物均以粗制的药水作为基础材料，与特定元素结合酿造而成。
类型限制：药物仅有普通的饮用版本，无法被改造成可投掷的喷溅或滞留型。

### 不可酿造的药水
以下药水不能通过酿造获得。在 [Java版] 中，幸运药水可以使用命令（如`/give @s minecraft:potion[minecraft:potion_contents="luck"]`）获得或在 [创造模式物品栏] 中获得。
以下药水具有特殊效果，但无法在生存模式中通过酿造台常规制作获得。
【不可合成的药水】
1. 版本： 仅 Java版
2. 延长版： 无
3. 增强版： 无
4. 效果： 无效果。这是一个没有任何状态效果的、无法通过酿造获得的特殊药水物品。
【幸运药水】
1. 版本： 仅 Java版 (5:00)
2. 延长版： 无
3. 增强版： 无
4. 效果： 幸运 I。此效果能将玩家的幸运属性提升1点，从而增加从钓鱼、开启战利品箱等依赖幸运值的活动中获得高质量战利品的概率。
5. 获取方式： 无法酿造，通常只能通过创造模式物品栏或命令获取。
【衰变药水】
1. 版本： 仅 基岩版 (0:40)
2. 延长版： 无
3. 增强版： 无
4. 效果： 凋零 I。此效果会使生物每秒受到1点（）伤害，与中毒效果不同，凋零效果可以致死。
5. 获取方式： 无法酿造，通常只能通过创造模式物品栏或命令获取。其效果类似于可通过命令获得的“凋零”状态效果。
【总结说明】：
共同特点： 这些药水均无法在生存模式的酿造台中制作。
版本独占：
幸运药水和不可合成的药水是Java版独有的。
衰变药水是基岩版独有的。
用途： 这些药水主要用于创造模式、地图制作或通过命令实现特定游戏效果，为玩家和地图作者提供了更多控制和自定义的可能性。


## Tips - 玩家可能不知道的知识
- 3个药水槽里只要有1个放有药水瓶，酿造过程就会继续，且可以继续放入药水瓶。然而，若移走酿造材料或清空药水槽，那么酿造过程就会终止。
- 使用红石粉来升级没有持续时间的药水（瞬间治疗，瞬间伤害）没有任何作用。
- 虽然Jeb说过在 [Beta 1.9-pre3] 里有161种药水组合而将来会有2,653种，但实际不使用外部程序的话只有22种不同的药水。这些药水中的其中19种各有8种不同效果。
- 喷溅药水能够用发射器发射并且可以激活木质压力板。
- [亡灵生物] 不受剧毒药水和再生药水的影响，但会受治疗药水的伤害，并从伤害药水中获得治疗，同时会受衰变药水的凋零效果影响。
- 衰变药水只有II级形式。
