# 附魔 - 魔咒
**魔咒（Enchantment）** 是适用于 [物品] 的额外效果，能够为物品提供多种功能。具有魔咒的物品被称为 **附魔物品（Enchanted Item）**。

## 机制
魔咒是普适于所有物品的机制，存储在物品的物品数据中。
所有物品都可以具有任意魔咒并生效，但为物品添加魔咒会受到许多因素的限制。
常见的可附魔物品包括 [盔甲]、 [工具]、 [武器]； [附魔书]、 [磁石指针] 等其他物品也可以获得一些魔咒。
具有魔咒的物品会获得 [稀有度] 提升和 [光效]。

### 魔咒属性
物品上的每一个魔咒仅有一个属性：
- **【等级】** 决定了许多魔咒的效果强度。
  - 魔咒数据能存储的最高等级是255\[仅 [Java版]\]或32767\[仅 [基岩版]\]。
魔咒有以下基本属性：
1. 魔咒的 **【适用物品】** 限定了哪些物品可被添加上该魔咒；该魔咒是这些物品的 **【适用魔咒】**，而这些物品也即该魔咒的 **【适用物品】**。
2. 魔咒的 **【可附魔物品】** 限定了哪些物品在经过附魔后可添加上该魔咒。该属性仅限定了该魔咒在“附魔”情境下所被附加到的载体，在其他情境下，通过某种方法，该魔咒可能被允许附加到任何载体上。
  - 绝大多数魔咒的可附魔物品与适用物品是相同的； [Java版] 有以下例外：
    - [锋利]、 [亡灵杀手]、 [节肢杀手] 排除了 [斧]。
    - [火焰附加]、 [亡灵杀手]、 [节肢杀手] 排除了 [重锤]。
    - [荆棘] 排除了 [头盔]、 [护腿]、 [靴子]。
- 魔咒不能与 **【不共存魔咒】** 被添加到同一个物品上。
- 魔咒的等级不能超过 **【最大等级】**。
  - 魔咒的最大等级为I级时，I级的该魔咒不会在 [提示框] 中显示等级。
- 魔咒的 **【出现条件】** 限制了为物品添加该魔咒的方式。
  - **【宝藏魔咒】** 意味着附魔台、村民售卖的装备、生物天然装备的附魔过程不能提供该魔咒。
  - **【无法交易获得】** 意味着村民的交易选项不能提供具有该魔咒的物品。
  - **【无法以战利品获得】** 意味着战利品生成（如 [箱子战利品] 和 [钓鱼]）的附魔过程不能提供该魔咒。
    - 部分战利品会直接生成包含指定魔咒的物品，不使用附魔过程，从而使该魔咒可以通过战利品获取。
- 魔咒的每个等级的 **【修正附魔等级范围】** 由 **【最小修正附魔等级】** 和 **【最大修正附魔等级】** 构成，决定了在 [附魔] 过程中，根据附魔操作的修正附魔等级，该魔咒的这个等级是否会可以被附魔选中；最小修正附魔等级还决定了 [砂轮] 移除该魔咒时提供的 [经验值]。
  - 一部分魔咒的高等级具有极高的最小修正附魔等级，导致物品无法通过附魔台的30级附魔得到它们，如 [锋利] V和 [荆棘] III，但其他方式（如 [末地城] 战利品的39级附魔、村民交易选项、或通过 [铁砧] 直接添加魔咒）仍然可以提供这些魔咒。
  - 在 [Java版] 中， [快速装填] III的修正附魔等级范围为52–50，因此也无法通过附魔获得。
- 魔咒的 **【挑选权重】** 决定了在附魔过程中，根据修正附魔等级决定了可选魔咒后，魔咒最终会被选中的相对权重。
- 魔咒的 **【合并魔咒花费】** 决定了在铁砧上为其他物品添加该魔咒需要消耗的 [经验等级]。
- 少数魔咒是 **【诅咒】**，在提示框中的文本颜色为红色，且不能通过通常的方式从物品上移除。

### 附魔书
[附魔书] 是专门用于存储魔咒的物品。所有魔咒都可以被添加到附魔书上，无论其适用物品。
附魔书的主要用途是在 [铁砧] 上作为牺牲物品来为目标物品添加魔咒。
在 [铁砧] 中使用附魔书来获得同样的魔咒所需要的经验消耗要比结合物品少得多。但因为制作附魔书本身也需要消耗经验等级，且附魔书的附魔能力不高，而直接附魔物品有概率同时得到多个魔咒，所以 **不建议** 有意制作附魔书后将其与物品合并。
附魔书的最大优势在于可以将针对某类工具的魔咒囤积起来日后使用，或从 [图书管理员] 购买大量需要的魔咒。而且附魔书可以进行 **受控** 的魔咒组合，例如，“精准采集”附魔书可以为斧、镐或锹附魔，玩家可以随意选择将该附魔书用在哪一种工具上。
附魔书存储魔咒的实现与其他具有魔咒的物品有诸多不同：
- 在 [Java版] 中，附魔书使用自己专有的物品数据来存储魔咒，物品本身并不是附魔物品，因此附魔书存储的魔咒不会产生效果。
- 在 [基岩版] 中，附魔书直接具有魔咒。大多数魔咒都不在附魔书上生效，但 [抢夺]、 [时运] 会在一些情况下生效，如可以使用具有抢夺的附魔书击杀生物以获取更多掉落物。

## 获取附魔物品
获取附魔物品主要有下面几个途径:【自然生成】，【附魔】，【铁砧】，【物品修复】和【作弊与命令】。
### 【自然生成】
许多物品在生成时就具有魔咒：
以下途径获得的物品具有单一的魔咒。这些途径生成的魔咒不使用 [附魔（物品修饰）] 机制。若无特殊说明，则所有魔咒出现的概率相同，可以施加的宝藏型魔咒有 [经验修补]、 [冰霜行者]、 [绑定诅咒] 和 [消失诅咒]，魔咒出现概率与魔咒的等级无关。
以下是自然生成魔咒的可能事件：
1. 搜寻 [废弃矿井]、 [地牢]、 [沙漠神殿]、 [掠夺者前哨站]、 [林地府邸]、大型 [海底废墟]、 [远古城市] 等 [结构] 中的 [箱子]，获得其中的 [附魔书]。
2. [图书管理员] 村民的许多 [交易] 选项会提供附魔书。
3. [废弃传送门] 的箱子中的金质装备、 [试炼密室] 柱廊木桶中的 [铁斧]、 [试炼密室] 交叉口木桶中的 [钻石斧] 附有随机魔咒。
4. 在袭击中生成的 [掠夺者] 携带的 [弩] 可能会附有 [快速装填] I（第4-5波）或 [快速装填] II（第6-8波）；袭击中生成的 [卫道士] 携带的 [铁斧] 可能会附有 [锋利] I（第1-5波）或 [锋利] II（第6-8波）。
5. 搜寻 [堡垒遗迹] 结构中的箱子，获得附有随机等级的 [灵魂疾行] 的附魔书。
6. 使用 [金锭] 与 [猪灵] 进行 [以物易物] 有概率获得附有随机等级的 [灵魂疾行] 的 [铁靴子] 或 [附魔书]。
7. 搜寻 [远古城市] 结构中的箱子，获得附有随机等级的 [迅捷潜行] 的附魔书。
8. 解锁 [试炼密室] 结构中的 [不祥宝库]，获取附有 [风爆] I的附魔书。
9. 解锁 [试炼密室] 结构中的 [宝库] 或搜寻其中的奖品箱，获得附有特定魔咒、等级随机的附魔书。
  - [宝库] 和 [不祥宝库] 中的附魔书均可能附有 [激流]、 [忠诚]、 [引雷]、 [穿刺] 或 [经验修补]。
  - [宝库] 中的附魔书还可能附有 [锋利]、 [节肢杀手]、 [效率]、 [时运]、 [精准采集] 或 [摔落缓冲]。
  - [不祥宝库] 中的附魔书还可能附有 [破甲]、 [致密]。
以下途径获得的物品可能会具有多个魔咒。这些途径生成的魔咒使用 [附魔（物品修饰）] 机制，且 [附魔等级] 各不相同。通常来说，附魔等级越高，魔咒的等级越高，魔咒的数量也越多。
1. 部分生物在生成时会持有附魔物品（如 [僵尸]、 [尸壳]、[僵尸村民]\[仅 [Java版]\]、 [溺尸]、 [卫道士]、 [掠夺者]、 [骷髅]、 [流浪者]、 [沼骸]、 [僵尸猪灵] 和 [猪灵]）；杀死生物有机会使其天然装备掉落，获取其附魔物品。这些物品可以进行5-22级附魔，具体等级取决于 [副区域难度]，不能附有宝藏型魔咒。
2. [盔甲匠] 村民售卖的 [钻石盔甲]、 [渔夫] 村民售卖的 [钓鱼竿]、 [制箭师] 村民售卖的 [弓] 和 [弩]、 [工具匠] 村民售卖的铁质或钻石质 [工具]、 [武器匠] 村民售卖的铁质或钻石质 [武器]、流浪商人售卖的 [铁镐]。这些物品可以进行5-19级附魔，不能附有宝藏型魔咒。
3. [钓鱼] 有概率得到附魔的 [钓鱼竿]、 [弓]、 [附魔书]。这些物品可以进行30级附魔，可以附有宝藏型魔咒。
4. 搜寻 [要塞]、 [丛林神庙]、 [末地城]、 [远古城市]、 [试炼密室] 等 [结构] 中的 [箱子]，获取其 [箱子战利品]。
  - [要塞]、 [丛林神庙] 的箱子中的 [附魔书] 可以进行30级附魔。可以附有宝藏型魔咒。
  - [末地城] 的箱子中的铁质、钻石质 [装备] 和 [远古城市] 的箱子中的 [铁护腿] 可以进行20–39级附魔。可以附有宝藏型魔咒。
  - [试炼密室] 的箱子战利品和 [宝库] 战利品中的铁胸甲、铁斧可以进行0–10级附魔， [弓] 可以进行5-15级附魔，弩可以进行5-20级附魔，钻石胸甲、钻石斧可以进行5-15级附魔。特别地， [不祥宝库] 中的钻石胸甲、钻石斧可以进行10-20级附魔，无附魔的弓和铁质装备。可以附有宝藏型魔咒。
  - [远古城市] 的箱子战利品中的钻石锄、钻石护腿可以进行30–50级附魔。可以附有宝藏型魔咒。
### 【附魔】
**【附魔（Enchanting）】** ：是一种为不具有魔咒的物品随机生成一个或多个魔咒的方式。
附魔得到的魔咒取决于参数 **【附魔等级】** 和物品的属性 **【附魔能力】**。一般来说，这两个参数越大，物品得到的附魔品质趋于更好，数量趋于更多。
[附魔台] 提供了玩家可操作的附魔方式，通过消耗 [青金石] 和 [经验等级]，可以为大多数 **【没有魔咒】** 的物品添加一个或数个魔咒。为了提供更多和高等级的魔咒，附魔台还需要在附近设置 [书架]。
然而，附魔台无法为物品添加 [宝藏型魔咒] 或最小修正附魔等级过高的魔咒，如钻石剑无法被添加 [锋利] V，钻石胸甲无法被添加 [荆棘] III等。某魔咒的适用物品，但不是可附魔物品，无法添加该魔咒，详见上文。一些没有附魔能力的物品，如 [盾牌]、 [剪刀]、 [刷子]、 [打火石]、 [胡萝卜钓竿]、 [诡异菌钓竿]、 [鞘翅] 等无法使用附魔的方式添加魔咒。这些情况下，玩家可以使用 [铁砧] 为物品添加魔咒。
附魔台、多数附魔战利品、生物天然装备均使用附魔来为物品添加魔咒。 [书] 可以通过附魔变为 [附魔书]，用于在 [铁砧] 上提供魔咒。
### 【铁砧】
[铁砧] 通过消耗 [经验等级] 和一个 **【牺牲物品】**，可以为一个物品（称作 **【目标物品】**）添加牺牲物品的魔咒。
在铁砧上使用的牺牲物品必须和目标物品完全相同（例如，铁镐和钻石镐不能合并）或者是一个 [附魔书]。牺牲物品是附魔书时，合并魔咒需要的经验等级更少。
用附魔为物品添加魔咒的方式存在随机性和局限性，通常难以满足玩家的需求。此时通过铁砧操作来为物品添加魔咒是必要的。然而，即使使用铁砧，也无法为目标物品添加不适用的魔咒或与原有魔咒不共存的魔咒。
在铁砧上进行操作会为物品增加“累积惩罚”，导致物品在铁砧上进行操作需要消耗的经验等级大量增加。 [生存模式] 中，当一次操作需要的经验等级大于等于40级时，铁砧会拒绝进行此操作，并提示“过于昂贵！”， [创造模式] 则无视此限制。因此，玩家要合理规划自己的操作以节省经验等级。
在 [Java版] 中，在创造模式中使用 [铁砧] 结合魔咒可以让魔咒被添加到不适用的物品上，但仍不能添加与原有魔咒不共存的魔咒。
### 【物品修复】
通过 [合成] 和 [砂轮] 进行的 [物品修复] 会合并两个物品的魔咒，不过绝大多数魔咒都会被物品修复移除，因此物品修复只能用于合并两个物品的诅咒魔咒。
### 【作弊与命令】
在 [创造模式物品栏] 中可以获取所有魔咒的对应 [附魔书]。
1. `/enchant`可以为执行者的主手物品添加魔咒，且受到魔咒的适用物品、最大等级、不共存魔咒的限制。
2. `/give`在给予玩家物品时，可以通过指定物品的`enchantments`或`stored_enchantment` [数据组件] 来指定该物品的魔咒。该方法不受以上限制。
3. `/item`可以通过某些 [物品修饰器] 来修改方块或生物的槽位中的物品具有的魔咒，或为物品添加随机等级、随机种类的魔咒，或使用 [附魔（物品修饰）] 机制进行附魔。该方法不受以上限制。
4. `/data`可以通过修改方块（如箱子、熔炉）或生物 [物品栏] 的某 [槽位] 中的物品的数据组件修改该物品的魔咒。该方法不受以上限制。


## 移除物品魔咒
物品上所有不是诅咒的魔咒可以通过以下方式被移除：
1. 通过 [合成] 来合并两个物品。
2. 通过 [砂轮] 进行祛魔或合并两个物品，并根据移除的魔咒的修正附魔等级提供 [经验值]。
[方块] 被放置后损失其所有魔咒。

## 所有魔咒
以下是Minecraft中各种魔咒的名称、最高等级、效果描述及适用物品的完整列表。
其中介绍的附魔魔咒为其名称，后面的罗马数字是该魔咒所能达到的最高等级。
【武器与战斗类魔咒】
1. 【节肢杀手】 (V)
- 描述： 对节肢生物（如蜘蛛、蜜蜂）造成额外伤害并附加缓慢效果。
- 适用物品： 剑、斧、矛 (新增于 JE 1.21.11)。
2. 【破甲】 (IV)
- 描述： 降低攻击目标的护甲值对攻击伤害的减少效果。
- 适用物品： 重锤。
3. 【火焰附加】 (II)
- 描述： 使被攻击目标着火。
- 适用物品： 剑、矛 (新增于 JE 1.21.11)。
4. 【锋利】 (V)
- 描述： 增加近战攻击伤害。
- 适用物品： 剑、斧、矛 (新增于 JE 1.21.11)。
5. 【亡灵杀手】 (V)
- 描述： 对亡灵生物（如僵尸、骷髅）造成额外伤害。
- 适用物品： 剑、斧、矛 (新增于 JE 1.21.11)。
6. 【击退】 (II)
- 描述： 增加击退距离。
- 适用物品： 剑、矛 (新增于 JE 1.21.11)。
7. 【抢夺】 (III)
- 描述： 增加生物掉落物的数量或概率。
- 适用物品： 剑、矛 (新增于 JE 1.21.11)。
8. 【横扫之刃】 (III) [仅 Java版]
- 描述： 增加横扫攻击伤害。
- 适用物品： 剑。
【远程武器与弹射物类魔咒】
1. 【力量】 (V)
- 描述： 增加弓箭伤害。
- 适用物品： 弓。
2. 【冲击】 (II)
- 描述： 增加弓箭的击退距离。
- 适用物品： 弓。
3. 【火矢】 (I)
- 描述： 使射出的箭矢着火。
- 适用物品： 弓。
4. 【无限】 (I)
- 描述： 射箭时不会消耗普通箭矢。

### 宝藏型魔咒
宝藏型魔咒是无法从附魔台等常见来源获取的魔咒。若要将这些魔咒应用到玩家预期的物品上，唯一的方式是获取拥有这些魔咒的物品，并使用[铁砧]来进行物品合并。不同的宝藏型魔咒有不同的获取方式：
主要有四类宝藏型魔咒，并提供了每一类附魔魔咒的具体获取途径：
1. [经验修补]、 [冰霜行者]、 [绑定诅咒]、 [消失诅咒]
- [交易] 以获取具有这些魔咒的工具或附魔书。图书管理员提供的具有宝藏型魔咒的附魔书，相较于其会提供的其他附魔书，价格会增加一倍。
- [钓鱼] 时获得的物品或附魔书有可能具有这些魔咒。
- 一部分 [箱子战利品] 会包含有概率出现这些魔咒的物品或附魔书。
- 在 [基岩版] 中， [袭击] 中生成的 [掠夺者] 和 [卫道士] 有可能会额外掉落具有宝藏型魔咒的附魔书。
2. [灵魂疾行]
- 进行 [以物易物] 有概率得到具有灵魂疾行的附魔书或铁靴子。
- 探索 [堡垒遗迹] 的战利品箱有概率找到灵魂疾行附魔书。
3. [迅捷潜行]
- 探索 [远古城市] 的战利品箱有概率找到具有迅捷潜行的附魔书。
4. [风爆]
- 解锁 [试炼密室] 的不祥宝库有概率找到具有风爆的附魔书。
